perl-Text-CHM

Perl extension for handling MS Compiled HtmlHelp Files

  • Bodhi
  • Bugzilla
  • FAF
  • Koji
  • SCM

Text::CHM is a module that implements a (partial) support for handling MS Compiled HtmlHelp Files (chm files for short) via CHMLib. Text::CHM allows you to open chm files, get their filelist, get the content of each file and close them; at the moment, no write support is available.
