Adobe OpenType monospaced font for mixed Latin and CJK text

Source Han Mono, which is derived from Source Han Sans and Source Code Pro, is an OpenType/CFF Collection (OTC) that includes 70 font instances—consisting of seven weights, five languages, and two styles—and is a Pan-CJK version of Source Han Code JP.
