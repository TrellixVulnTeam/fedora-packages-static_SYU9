xlcrack

Recover lost and forgotten passwords from XLS files

  • Builds
  • Updates
  • Bugs
  • Sources

  • Koschei

xlcrack recovers lost passwords for XLS files, such as those saved by Excel 95. In order to do this it implements a simple XOR encryption algorithm common to several Microsoft Office applications.
