Pure Rust implementation of Public-Key Cryptography Standards (PKCS) #8: Private-Key Information Syntax Specification (RFC 5208), with additional support for PKCS#8v2 asymmetric key packages (RFC 5958). This package contains library source intended for building other packages which use the "sha1" feature of the "pkcs8" crate.
