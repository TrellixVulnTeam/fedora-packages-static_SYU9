Adds i18n/l10n support to Flask applications with the help of the Babel library. This is fork of official Flask-Babel extension with following features: 1 - It is possible to use multiple language catalogs in one Flask application; 2 - Your extension can package localization file(s) and use them if necessary; 3 - Does not reload localizations for each request.
