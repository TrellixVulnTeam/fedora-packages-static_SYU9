Don't you love when you accidentally tap your Yubikey when you have your IRC client in focus and you send 987947 into Libera? Want to be able to have that experience without having to reach all the way over to your laptop's USB port? Now you can!. This package contains library source intended for building other packages which use the "default" feature of the "yubibomb" crate.
