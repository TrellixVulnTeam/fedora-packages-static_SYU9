JSON Lint for PHP. This library is a port of the JavaScript jsonlint (https://github.com/zaach/jsonlint) library. Bin: /usr/bin/jsonlint-php [1] Autoloader: /usr/share/php/Seld/JsonLint/autoload.php [1] Package python2-demjson installs /usr/bin/jsonlint so this package's bin script has been renamed to /usr/bin/jsonlint-php
