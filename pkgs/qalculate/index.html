Qalculate! is a multi-purpose desktop calculator for GNU/Linux. It is small and simple to use but with much power and versatility underneath. Features include customizable functions, units, arbitrary precision, plotting. This package provides the text-mode interface for Qalculate! The GTK and QT frontends are provided by qalculate-gtk and qalculate-kde packages resp.
