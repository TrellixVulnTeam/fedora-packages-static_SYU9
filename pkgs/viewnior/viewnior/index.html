Viewnior is an image viewer program. Created to be simple, fast and elegant. It's minimalistic interface provides more screen space for your images. Among its features are: * Fullscreen & Slideshow * Rotate, flip, save, delete images * Animation support * Browse only selected images * Navigation window * Simple interface * Configurable mouse actions
