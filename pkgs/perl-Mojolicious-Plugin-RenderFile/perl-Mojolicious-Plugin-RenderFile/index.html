perl-Mojolicious-Plugin-RenderFile

Mojolicious plugin allowing customization to force file download

Mojolicious::Plugin::RenderFile is a Mojolicious plugin that makes it easy to provide files for download. It also allows customization of the HTTP headers sent to the client.
