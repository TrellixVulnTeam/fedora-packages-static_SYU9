xmms-flac Subpackage of flac

XMMS plugin needed to play FLAC (Free Lossless Audio Codec) files

FLAC is a Free Lossless Audio Codec. The FLAC format supports streaming, seeking, and archival, and gives 25-75% compression on typical CD audio. This is the input plugin for XMMS to be able to read FLAC files.
