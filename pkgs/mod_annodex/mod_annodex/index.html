mod_annodex

Apache module for server-side support of annodex media

mod_annodex provides full support for Annodex.net media. For more details about annodex format, see http://www.annodex.net/ mod_annodex is a handler for type application/x-annodex. It provides the following features: * dynamic generation of Annodex media from CMML files. * handling of timed query offsets, such as http://media.example.com/fish.anx?t=npt:01:20.8 or http://media.example.com/fish.anx?id=Preparation * dynamic retrieval of CMML summaries, if the Accept: header prefers type text/x-cmml over application/x-annodex.
