ocaml-logs

Logging infrastructure for OCaml

  • Koschei

Logs provides a logging infrastructure for OCaml. Logging is performed on sources whose reporting level can be set independently. The log message report is decoupled from logging and is handled by a reporter. A few optional log reporters are distributed with the base library and the API lets you easily implement your own.
