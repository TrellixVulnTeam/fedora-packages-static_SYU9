clover2 is a Ruby-like compiler language with static type like Java. This language consists of compilers and virtual machines like Java and C#. In order to compile, type checking can be done at compile time. In addition, it is designed to be able to use an easy-to-use library like Ruby. Regular expressions, lambda, closure etc are first class objects.
