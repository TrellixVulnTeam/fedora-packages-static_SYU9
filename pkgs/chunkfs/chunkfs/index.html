chunkfs

FUSE based filesystem that allows you to mount an arbitrary file or block device

ChunkFS is a FUSE based filesystem that allows you to mount an arbitrary file or block device as a directory tree of files that each represent a chunk of user-specified size of the mounted file. The chunk size is global per mount, but at mount time any value can be specified. (If the file size isn't a multiple of the specified chunk size, the last file in the tree simply will be smaller than the chunk size.) Only read access is supported at the moment.
