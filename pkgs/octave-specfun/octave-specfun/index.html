octave-specfun

Special functions for Octave, including ellipitic functions

  • Builds

This package contains special functions for Octave, including elliptic functions, sine/cosine integral functions, complementary error functions and exponential integrals, Heaviside and Dirac functions, the Riemann zeta function and others.
