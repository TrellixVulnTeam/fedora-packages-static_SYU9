anope-ldap Subpackage of anope

LDAP modules for Anope IRC services

Anope is a set of IRC services designed for flexibility and ease of use. This package provides Anope modules to enable IRC commands such as IDENTIFY, RELEASE, RECOVER, GHOST, etc. to use LDAP to authenticate users. It provides a module to tie users to Anope opertypes when they identify via LDAP group membership, too.
