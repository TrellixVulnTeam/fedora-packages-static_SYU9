Manage configuration for a command-line tool. This component is designed to provide the components needed to manage configuration options from different sources, including: * Commandline options * Configuration files * Alias files (special configuration files that identify a specific target site) * Default values (provided by command) Symfony Console is used to provide the framework for the command-line tool, and the Symfony Configuration component is used to load and merge configuration files. This project provides the glue that binds the components together in an easy-to-use package. Autoloader: /usr/share/php/Consolidation/Config/autoload.php
