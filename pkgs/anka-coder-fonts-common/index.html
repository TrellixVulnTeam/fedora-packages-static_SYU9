The Anka/Coder family is a mono spaced, courier-width (60% of height; em size 2048x1229) font that contains characters from 437, 866, 1251, 1252 and some other code pages and can be used for source code, terminal windows etc. There are 3 font sets (regular. italic, bold, bold-italic each): 1. Anka/Coder (em size 2048x1229) 2. Anka/Coder Condensed (condensed by 12.5%; em size 2048x1075) 3. Anka/Coder Narrow (condensed by 25%; em size 2048x922) This package consists of files used by other anka-coder-fonts packages.
