This package provides bindings to the 'gettext' internationalization and localization (i18n) library. This package provides support for custom 'Setup.hs' scripts via the "Distribution.Simple.I18N.GetText" module. A user-contributed tutorial can be found in the [Haskell Wiki](https://wiki.haskell.org/Internationalization_of_Haskell_programs_using_gettext).
