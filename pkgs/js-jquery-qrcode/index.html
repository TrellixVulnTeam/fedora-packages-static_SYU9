js-jquery-qrcode

A JavaScript library for standalone qrcode generation

jquery.qrcode.js is jquery plugin for a pure browser qrcode generation. It allows you to easily add qrcode to your web pages. It is standalone, and less than 4 kB after minify+gzip. It doesn't rely on external services which go on and off, or add latency while loading. It is based on qrcode-generator which builds qrcodes in various language. jquery.qrcode.js wraps that library to make it easy to include in your website. Note: The upstream version of this library bundles qrcode-generator into this package, but Fedora has chosen to unbundle it. Thus, you will need to include qrcode.js from js-qrcode-generator in a script tag before including this script if you wish to use this library.
