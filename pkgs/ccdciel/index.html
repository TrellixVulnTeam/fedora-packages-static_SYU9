CCDciel is a free CCD capture software intended for the amateur astronomer. It include all the features required to perform digital imaging CCD observation of celestial objects. Using the standard drivers protocol INDI and ASCOM it can connect and control the CCD camera, the focuser, the filter wheel and the telescope mount. It tightly integrates with Skychart to provide telescope control while Indistarter can be used to control INDI server drivers
