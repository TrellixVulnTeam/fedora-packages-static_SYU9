AllegroOGG

Ogg library for use with the Allegro game library

  • Bodhi
  • Bugzilla
  • FAF
  • Koji

AllegroOGG is an Allegro wrapper for the Ogg Vorbis decoder from the Xiph.org foundation. This lib lets you play OGGs and convert OGGs to Allegro SAMPLEs amongst a lot of other capabilites.
