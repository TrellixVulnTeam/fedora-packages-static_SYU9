gdouros-textfonts-doc Subpackage of gdouros-alexander-fonts

Documentation for all Textfonts by G. Douros

This package contains documentation regarding the Textfonts family of fonts by G. Douros, i.e. Aroania, Anaktoria, Alexander, Avdira and Asea. The origin of each font is presented, as well as sample texts along with a character overview and opentype features supported by the fonts.
