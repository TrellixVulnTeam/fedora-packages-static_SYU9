sip-redirect

Tiny IPv4 and IPv6 SIP redirect server written in Perl

  • Builds

sip-redirect is a tiny SIP redirect server written in Perl. It is IPv4 and IPv6 capable, but the IPv6 support is optional. The RFC 3261 was the base for this simple and very configurable implementation. There is neither TCP nor multicast support programmed in.
