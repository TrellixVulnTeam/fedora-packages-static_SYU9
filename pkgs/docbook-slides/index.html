DocBook Slides provides customization layers of the both the Simplified and the full DocBook XML DTD, as well as the DocBook XSL Stylesheets. This package contains the XML document type definition and stylesheets for processing DocBook Slides XML. The slides doctype and stylesheets are for generating presentations, primarily in HTML.
