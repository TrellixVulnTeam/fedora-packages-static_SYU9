chordpro

ChordPro will read a text file containing the lyrics of one or many songs plus chord information. ChordPro will then generate a photo-ready, professional looking, impress-your-friends sheet-music suitable for printing on your nearest printer. To learn more about ChordPro, look for the man page or do chordpro --help for the list of options. ChordPro is a rewrite of the Chordii program. For more information about ChordPro, see https://www.chordpro.org.
