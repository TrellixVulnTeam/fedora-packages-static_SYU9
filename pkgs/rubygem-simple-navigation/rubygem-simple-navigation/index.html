rubygem-simple-navigation

Ruby library for creating navigation for your Rails or Sinatra application

  • Builds
  • Updates
  • Bugs

  • Crash Reports
  • Koschei

With the simple-navigation gem installed you can easily create multilevel navigation for your Rails, Sinatra or Padrino applications. The navigation is global in a single configuration file. It supports automatic as well as explicit highlighting of the currently active navigation through regular expressions.
