Subliminal is a Python library to search and download subtitles. It comes with an easy to use yet powerful CLI suitable for direct use or cron jobs. Subliminal uses multiple providers to give users a vast choice and have a better chance to find the best matching subtitles. Current supported providers are: - Addic7ed - LegendasTV - NapiProjekt - OpenSubtitles - Podnapisi - Shooter - TheSubDB - TvSubtitles
