libreoffice-TexMaths

A LaTex Equation Editor for LibreOffice

TexMaths is a LaTeX equation editor for LibreOffice. It is derived from OOoLatex, originally developed by Geoffroy Piroux. As its predecessor, TexMaths is a LibreOffice extension that allows you to enter and edit LaTeX equations directly into LibreOffice documents.
