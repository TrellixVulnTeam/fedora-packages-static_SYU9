Changelog

Date Author Change
2021-04-02 Audrey Toskin - 27-7.20210319.295026a - Recommend also installing GNOME Shell extension for libappindicator. (TopIcons Plus and libappindicator provide overlapping functionality and use case, but cover the gaps in each other's supported apps and sessions, without stepping on each other's toes. So a user who wants to see status icons in the top panel will probably want both Shell extensions.)
