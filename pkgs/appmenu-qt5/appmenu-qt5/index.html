appmenu-qt5

Support for global DBus-exported application menu in Qt5

This is a different, Qt5-compatible approach of the existing appmenu-qt (https://launchpad.net/appmenu-qt). appmenu-qt5 is a Qt5 QPA theme plugin that adds support for application menus to Qt5 applications. This only works for Qt5 versions >= 5.2 currently. To enable the support, set QT_QPA_PLATFORMTHEME=appmenu-qt5 in your environment or install the appmenu-qt5-profile.d package to enable system-wide, see README.fedora *BEFORE* for further information.
