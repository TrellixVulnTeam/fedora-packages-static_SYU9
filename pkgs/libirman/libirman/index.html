libirman

Library for IRMAN hardware

  • Builds

Runtime libraries for accessing the IrMan hardware. The IrMan hardware((http://www.intolect.com/irmandetail.htm) is nowadays discontinued. However, some modern hardware (notably the irtoy) is able to emulate the irman protocol.
