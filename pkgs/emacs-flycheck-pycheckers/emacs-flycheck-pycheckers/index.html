emacs-flycheck-pycheckers

Multiple syntax checker for Python in Emacs, using Flycheck

This package provides a way to run multiple syntax checkers on Python code, in parallel. The list of supported checkers includes: * pylint * flake8 * pep8 * pyflakes * mypy * bandit This is an alternative way of running multiple Python syntax checkers in Flycheck that doesn't depend on Flycheck's chaining mechanism. Flycheck is opinionated about what checkers should be run, and chaining is difficult to get right. This package assumes that the user knows what they want, and can configure their checkers accordingly â€” if they want to run both flake8 and pylint, that's fine. This also allows us to run multiple syntax checkers in parallel, rather than sequentially.
