This package contains a Sphinx extension for creating document components optimized for HTML+CSS. - The panels directive creates panels of content in a grid layout, utilizing both the Bootstrap 4 grid system, and cards layout. - The link-button directive creates a clickable button, linking to a URL or reference, and can also be used to make an entire panel clickable. - The dropdown directive creates content that can be toggled. - The tabbed directive creates tabbed content. - opticon and fa (fontawesome) roles allow for inline icons to be added.
