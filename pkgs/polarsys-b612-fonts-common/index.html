Commissioned by Airbus and designed by Intactile Design, B612 is a digital font intended to be used in an aeronautical context. B612 is built with legibility as its core: every character is designed to be highly recognizable even in critical reading conditions. B612 drawing has been optimized for screen display, and full hinting has been added to all sizes of alpha numeric characters. This package consists of files used by other polarsys-b612-fonts packages.
