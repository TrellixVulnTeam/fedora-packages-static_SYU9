A Python interface to the Deutsche Post Internetmarke Online Franking

This is a python module for interfacing the "Internetmarke" API provided by the German postal company "Deutsche Post". It implements V3 of this API. The Internetmarke API allows you to buy online franking for national and international postal products like post cards and letters of all weight classes and service classes (normal, registered, ...).
