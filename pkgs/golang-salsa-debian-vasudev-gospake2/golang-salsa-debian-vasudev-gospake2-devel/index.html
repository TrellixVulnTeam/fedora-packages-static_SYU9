Implementation of SPAKE2 key exchange protocol which interoperates with Rust Haskell and Python versions. This package defines the behavior of group and its element as package groups. It also implements 2 groups ed25519 and multiplicative group over integer as 2 packages. SPAKE2 calculation uses ed25519 as default group and allows user to switch to group of his choice.
