A featureful ncurses based MPD client inspired by ncmpc. The main features are: - tag editor - playlist editor - easy to use search engine - media library - music visualizer - ability to fetch artist info from last.fm - new display mode - alternative user interface - ability to browse and add files from outside of MPD music directory .. and a lot more minor functions.
