Yaru theme is the default theme for Ubuntu, entirely backed by the community. This is the theme that is shaped by the community on the Ubuntu hub, turned into the default theme starting from Ubuntu 18.10 Cosmic Cuttlefish. This package contains GNOME Shell Theme.
