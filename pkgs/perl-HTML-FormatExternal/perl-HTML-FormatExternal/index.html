perl-HTML-FormatExternal

HTML to text formatting using external programs

  • Builds
  • Updates
  • Bugs
  • Sources
  • Crash Reports
  • Koschei

This is a common base for formatter modules which turn HTML into plain text by dumping it through the respective external programs. Available modules are: HTML::FormatText::Elinks HTML::FormatText::Html2text HTML::FormatText::Links HTML::FormatText::Lynx HTML::FormatText::W3m
