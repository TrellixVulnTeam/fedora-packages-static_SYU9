php-opencloud/openstack is an SDK which allows PHP developers to easily connect to OpenStack APIs in a simple and idiomatic way. This binding is specifically designed for OpenStack APIs, but other provider SDKs are available. Multiple OpenStack services, and versions of services, are supported. Autoloader: /usr/share/php/OpenStack/autoload.php
