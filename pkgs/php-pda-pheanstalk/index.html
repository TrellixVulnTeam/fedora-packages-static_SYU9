Pheanstalk is a pure PHP 5.3+ client for the beanstalkd workqueue [1]. It has been actively developed, and used in production by many, since late 2008. Created by Paul Annesley [2], Pheanstalk is rigorously unit tested and written using encapsulated, maintainable object oriented design. Community feedback, bug reports and patches has led to a stable 1.0 release in 2010, a 2.0 release in 2013, and a 3.0 release in 2014. Pheanstalk 3.0 introduces PHP namespaces, PSR-1 and PSR-2 coding standards, and PSR-4 autoloader standard. beanstalkd up to the latest version 1.4 is supported. All commands and responses specified in the protocol documentation for beanstalkd 1.3 are implemented. Autoloader: /usr/share/php/Pheanstalk/autoload.php [1] http://xph.us/software/beanstalkd/ [2] http://paul.annesley.cc/
