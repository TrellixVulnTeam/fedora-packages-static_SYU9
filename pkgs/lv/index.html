lv

A Powerful Multilingual File Viewer

  • Bodhi
  • Bugzilla
  • FAF
  • Koji
  • SCM

lv is a powerful file viewer like less. lv can decode and encode multilingual streams through many coding systems: ISO-8859, ISO-2022, EUC, SJIS, Big5, HZ, Unicode. It recognizes multi-bytes patterns as regular expressions, lv also provides multilingual grep. In addition, lv can recognize ANSI escape sequences for text decoration.
