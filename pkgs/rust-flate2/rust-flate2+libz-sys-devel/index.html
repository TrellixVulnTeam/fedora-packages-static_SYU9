DEFLATE compression and decompression exposed as Read/BufRead/Write streams. Supports miniz_oxide, miniz.c, and multiple zlib implementations. Supports zlib, gzip, and raw deflate streams. This package contains library source intended for building other packages which use "libz-sys" feature of "flate2" crate.
