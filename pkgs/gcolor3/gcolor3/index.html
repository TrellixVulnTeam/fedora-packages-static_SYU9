gcolor3

A simple color chooser written in GTK3 (like gcolor2)

Gcolor3 is a color selection dialog written in GTK+ 3. It is much alike Gcolor2, but uses the newer GTK+ version to better integrate into your modern desktop. It has the same feature set as Gcolor2, except that recent versions of Gcolor3 use an .ini style file to save colors (older versions use the same file as Gcolor2).
