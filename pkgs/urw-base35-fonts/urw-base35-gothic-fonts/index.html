This package contains URW Gothic font family, which is part of Level 2 Core Font Set. The Level 2 Core Font Set is a PostScript specification of 35 base fonts that can be used with any PostScript file. These fonts are provided freely by (URW)++ company, and are mainly utilized by applications using Ghostscript.
