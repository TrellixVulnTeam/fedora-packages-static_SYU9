gdouros-aegean-fonts

A font for ancient scripts in the greater Aegean vicinity

  • Bodhi
  • Bugzilla
  • FAF
  • Koji
  • SCM

Aegean covers the following scripts and symbols: Basic Latin, Greek and Coptic, Greek Extended, some Punctuation and other Symbols, Linear B Syllabary, Linear B Ideograms, Aegean Numbers, Ancient Greek Numbers, Ancient Symbols, Phaistos Disc, Lycian, Carian, Old Italic, Ugaritic, Old Persian, Cypriot Syllabary, Phoenician, Lydian, Archaic Greek Musical Notation, Cretan Hieroglyphs, Cypro-Minoan, Linear A, the Arkalochori Axe, Ancient Greek and Old Italic variant alphabets. Those of the above that are not supported by the Unicode Standard 8.0, they are allocated in the Supplementary Private Use Plane 15. It was created by George Douros.
