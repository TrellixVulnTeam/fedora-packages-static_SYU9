gnome-shell-extension-openweather

Display weather information from many locations in the world

gnome-shell-extension-openweather is an extension to display weather information from https://openweathermap.org or https://darksky.net for (almost) all locations of the world in GNOME Shell. Be aware, that system-wide installed gnome-shell-extensions are disabled by default and have to be enable by the user(s), if they get installed the first time. You can use gnome-tweak-tool (additional package) or run in terminal: gnome-shell-extension-tool -e openweather-extension@jenslody.de
