BDD style code blocks for PHPUnit / Codeception Specify allows you to write your tests in more readable BDD style, the same way you might have experienced with Jasmine. Inspired by MiniTest of Ruby now you combine BDD and classical TDD style in one test. Autoloader: /usr/share/php/Codeception/Specify/autoload.php
