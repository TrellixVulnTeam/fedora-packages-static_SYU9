perl-Net-Google-AuthSub

Provides interface to interact with sites that implement Google style AuthSub

AuthSub is Google's method of authentication for their web services. It is also used by other web sites. You can read more about it here. http://code.google.com/apis/accounts/Authentication.html A Google Group for AuthSub is here. http://groups.google.com/group/Google-Accounts-API
