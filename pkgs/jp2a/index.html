jp2a

Small utility that converts JPG images to ASCII (text) using libjpeg

  • Bodhi
  • Bugzilla
  • FAF
  • Koji
  • SCM

jp2a is a simple JPEG to ASCII converter. jp2a is very flexible. It can use ANSI colors and html in output. jp2a can also download and convert images from Internet via command line.
