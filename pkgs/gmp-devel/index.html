gmp-devel Subpackage of gmp

Development tools for the GNU MP arbitrary precision library

  • Bodhi

The libraries, header files and documentation for using the GNU MP arbitrary precision library in applications. If you want to develop applications which will use the GNU MP library, you'll need to install the gmp-devel package. You'll also need to install the gmp package.
