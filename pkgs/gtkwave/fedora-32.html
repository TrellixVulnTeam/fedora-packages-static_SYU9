2019-03-22 Paul Howarth - 3.3.100-1 - Update to 3.3.100 - FSDB fix for variable declarations of array of reals - Added Real, Time, Enum, and Popcnt flags to Edit/Show-Change - Ensure Show-Change regenerates analog traces - Added braces inside Tcl source command to allow spaces in filenames for Tcl scripts
2019-02-10 Paul Howarth - 3.3.99-1 - Update to 3.3.99 - Added visible single bit glitches as a yellow dot (if enabled with --rcvar 'vcd_preserve_glitches on') - Fixed print routine broken by bsearch_trunc() optimization in version 3.3.96
