Python (2 and 3) bindings for the WebHDFS (and HttpFS) API, supporting both secure and insecure clusters. Command line interface to transfer files and start an interactive client shell, with aliases for convenient name-node URL caching. Additional functionality through optional extensions: Avro, to read and write Avro files directly from HDFS. data-frame, to load and save Pandas data-frames. Kerberos, to support Kerberos authenticated clusters.
