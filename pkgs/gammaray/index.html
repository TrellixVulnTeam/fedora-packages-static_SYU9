gammaray

A tool for examining internals of Qt applications

  • Bodhi
  • Bugzilla
  • FAF
  • Koji
  • SCM

A tool to poke around in a Qt-application and also to manipulate the application to some extent. It uses various DLL injection techniques to hook into an application at run-time and provide access to a lot of interesting information. By default GammaRay can only introspect Qt 5 applications.
