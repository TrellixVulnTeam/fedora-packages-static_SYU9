tack

Terminfo action checker

  • Builds
  • Updates
  • Bugs
  • Sources
  • Crash Reports
  • Koschei

The tack program has three purposes: to help you build a new terminfo entry describing an unknown terminal, to test the correctness of an existing entry, and to develop the correct pad timings needed to ensure that screen updates don't fall behind the incoming data stream.
