Library and middleware enabling cross-origin resource sharing for your http-{foundation,kernel} using application. It attempts to implement the W3C Candidate Recommendation [1] for cross-origin resource sharing. Autoloader: /usr/share/php/Asm89/Stack/autoload-cors.php [1] http://www.w3.org/TR/cors/
