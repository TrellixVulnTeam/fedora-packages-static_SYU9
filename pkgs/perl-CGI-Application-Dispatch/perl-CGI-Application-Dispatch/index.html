perl-CGI-Application-Dispatch

Dispatch requests to CGI::Application based objects

  • Builds
  • Updates
  • Bugs
  • Sources
  • Crash Reports
  • Koschei

This module provides a way (as a mod_perl handler or running under vanilla CGI) to look at the path (as returned by dispatch_path) of the incoming request, parse off the desired module and it's run mode, create an instance of that module and run it.
