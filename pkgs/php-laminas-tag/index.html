Laminas\Tag is a component suite which provides a facility to work with taggable Items. As its base, it provides two classes to work with Tags, Laminas\Tag\Item and Laminas\Tag\ItemList. Additionally, it comes with the interface Laminas\Tag\TaggableInterface, which allows you to use any of your models as a taggable item in conjunction with Laminas\Tag. Documentation: https://docs.laminas.dev/laminas-tag/
