foo2ddst is an open source printer driver for printers that use the DDST wire protocol for their print data, such as the Ricoh SP 112, or the Ricoh SP 201Nw. These printers are often erroneously referred to as winprinters or GDI printers. However, Microsoft GDI only mandates the API between an application and the printer driver, not the protocol on the wire between the printer driver and the printer. In fact, DDST printers are raster printers which happen to use a very efficient wire protocol. DDST is just one of many wire protocols that are in use today, such as Postscript, PCL, Epson, ZjStream, etc. Users of this package are requested to visit the author's web page at http://foo2oak.rkkda.com/ and consider contributing.
