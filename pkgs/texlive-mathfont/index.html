texlive-mathfont Subpackage of texlive

Use TrueType and OpenType fonts in math mode; compatible with XeTeX and LuaTeX

  • Bodhi

  • SCM

The mathfont package provides a flexible interface for changing the font of math mode characters. The package allows the user to specify a default unicode font for each of six basic classes of Latin and Greek characters, and it provides additional support for unicode alphanumeric symbols. Crucially, mathfont is compatible with both LuaLaTeX and XeLaTeX, and it provides several font-loading commands that allow the user to change fonts locally or for individual symbols within math mode.
