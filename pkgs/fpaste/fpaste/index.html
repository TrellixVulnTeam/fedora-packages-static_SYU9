fpaste

A simple tool for pasting info onto the Fedora community paste server

  • Builds

  • Koschei

It is often useful to be able to easily paste text to the Fedora Pastebin at http://paste.fedoraproject.org and this simple script will do that and return the resulting URL so that people may examine the output. This can hopefully help folks who are for some reason stuck without X, working remotely, or any other reason they may be unable to paste something into the pastebin. This is not a general client for paste servers. It will only ever support the paste server that the Fedora community is running.
