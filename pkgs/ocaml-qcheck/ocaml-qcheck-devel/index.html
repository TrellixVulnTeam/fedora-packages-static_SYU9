Qcheck enables checking invariants (properties of a type) over randomly generated instances of the type. It provides combinators for generating instances and printing them. The ocaml-qcheck-devel package contains libraries and signature files for developing applications that use the qcheck compatibility wrapper. New code should instead use ocaml-qcheck-alcotest-devel or ocaml-qcheck-ounit-devel.
