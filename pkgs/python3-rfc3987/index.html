Parsing and validation of URIs (RFC 3986) and IRIs (RFC 3987)

This module provides regular expressions according to RFC 3986 "Uniform Resource Identifier (URI): Generic Syntax" and RFC 3987 "Internationalized Resource Identifiers (IRIs)" , and utilities for composition and relative resolution of references. This package includes the Python 3 version of the module.
