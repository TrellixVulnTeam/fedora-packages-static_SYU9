The Terminus Font is a clean, fixed with bitmap font designed for long (8 and more hours per day) work with computers. I contains more than 1300 characters, covers about 120 language sets and supports ISO8859-1/2/5/7/9/13/15/16, Paratype-PT154/PT254, KOI8-R/U/E/F, Esperanto, and many IBM, Windows and Macintosh code pages, as well as the IBM VGA, vt100 and xterm pseudo graphic characters. The sizes present are 6x12, 8x14, 8x16, 10x18, 10x20, 11x22, 12x24, 14x28, and 16x32. The weights are normal and bold (except for 6x12), plus CRT VGA-bold for 8x14 and 8x16. This package contains the font variants to use with software still relying on legacy PCF font rendering methods (e.g. GNU Emacs 26).
