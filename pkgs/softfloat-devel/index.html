Berkeley SoftFloat is a software implementation of binary floating-point that conforms to the IEEE Standard for Floating-Point Arithmetic. The current release supports five binary formats: 16-bit half-precision, 32-bit single-precision, 64-bit double-precision, 80-bit double-extended- precision, and 128-bit quadruple-precision. The softfloat-devel package contains libraries and header files for developing applications that use softfloat.
