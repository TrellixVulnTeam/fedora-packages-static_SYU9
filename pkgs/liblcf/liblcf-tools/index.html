liblcf-tools Subpackage of liblcf

Programs for handling RPG Maker 2000/2003 game data

This package contains helper tools for working with RPG Maker 2000/2003 data files: - lcf2xml: converts RM2k data files to XML (and vice-versa) - lcfstrings: extracts all strings from an RM2k data file
