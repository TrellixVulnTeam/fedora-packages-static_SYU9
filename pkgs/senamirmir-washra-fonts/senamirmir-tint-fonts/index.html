A set of high quality Unicode fonts for the Geʼez (Ethiopic) script published by the Senamirmir project. They can be used to write Ethiopic and Eritrean languages (Amharic, Blin, Geʼez, Harari, Meʼen, Tigre, Tigrinya…). This package consists of the “Ethiopic Tint” font family.
