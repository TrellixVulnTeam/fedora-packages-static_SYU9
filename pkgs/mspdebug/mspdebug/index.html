mspdebug

Debugger and gdb proxy for MSP430 MCUs

  • Builds
  • Updates

  • Koschei

A a free debugger for use with MSP430 MCUs. It supports FET430UIF, eZ430, RF2500 and TI Chronos devices. It can be used as a proxy for gdb or as an independent debugger with support for programming, disassembly and reverse engineering.
