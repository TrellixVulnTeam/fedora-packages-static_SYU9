python3-ceph-argparse Subpackage of ceph

Python 3 utility libraries for Ceph CLI

  • Bodhi
  • Bugzilla
  • FAF
  • Koji

This package contains types and routines for Python 3 used by the Ceph CLI as well as the RESTful interface. These have to do with querying the daemons for command-description information, validating user command input against those descriptions, and submitting the command to the appropriate daemon.
