php-stecman-symfony-console-completion

Automatic BASH completion for Symfony Console based applications

  • Builds

This package provides automatic (tab) completion in BASH and ZSH for Symfony Console Component based applications. With zero configuration, this package allows completion of available command names and the options they provide. User code can define custom completion behaviour for argument and option values. Autoloader: /usr/share/php/Stecman/Component/Symfony/Console/BashCompletion/autoload.php
