Provides functions available in later versions of 'base' to a wider range of compilers, without requiring you to use CPP pragmas in your code. the suffix '.Repl.Batteries', which are distinct from anything in depends on compatibility packages (such as 'semigroups') to offer a wider support window than 'base-compat', which has no dependencies.
