Veusz is a 2D and 3D scientific plotting package, designed to create publication-ready vector PDF and SVG output. It features GUI, command-line, and scripting interfaces. Graphs are constructed from widgets, allowing complex layouts to be designed. Veusz supports plotting functions, data with error bars, keys, labels, stacked plots, ternary plots, vector plots, contours, images, shapes and fitting data. 3D point, surface, volume and function plots are also supported.
