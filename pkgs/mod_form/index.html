mod_form

Apache module that decodes data submitted from Web forms

  • Bodhi
  • Bugzilla
  • FAF
  • Koji
  • SCM

Utility to decode data submitted from Web forms. It deals with both GET and POST methods where the data are encoded using the default content type application/x-www-form-urlencoded. It does not decode multipart/form-data (file upload) forms: for those you should use mod_upload.
