perl-Template-Plugin-Cycle

Cyclically insert into a Template from a sequence of values

  • Builds

Sometimes, apparently almost exclusively when doing alternating table row backgrounds, you need to print an alternating, cycling, set of values into a template. Template::Plugin::Cycle is a small, simple, and hopefully DWIM solution to these sorts of tasks.
