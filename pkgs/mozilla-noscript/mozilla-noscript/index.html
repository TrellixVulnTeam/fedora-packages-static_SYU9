mozilla-noscript

JavaScript white list extension for Mozilla Firefox

  • Builds

The NoScript extension provides extra protection for Firefox. It allows JavaScript, Java, Flash and other plug-ins to be executed only by trusted web sites of your choice (e.g. your online bank) and additionally provides Anti-XSS protection.
