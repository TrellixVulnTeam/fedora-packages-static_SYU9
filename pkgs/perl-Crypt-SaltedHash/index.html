perl-Crypt-SaltedHash

Perl interface to functions that assist in working with salted hashes

The Crypt::SaltedHash module provides an object oriented interface to create salted (or seeded) hashes of clear text data. The original formalization of this concept comes from RFC-3112 and is extended by the use of different digital algorithms.
