tortoisehg-nautilus Subpackage of tortoisehg

Mercurial GUI plug-in to the Nautilus file manager

  • Builds

This package contains the TortoiseHg Gnome/Nautilus extension, which makes the Mercurial distributed revision control system available in the file manager with a graphical interface. Note that the nautilus extension has been deprecated upstream.
