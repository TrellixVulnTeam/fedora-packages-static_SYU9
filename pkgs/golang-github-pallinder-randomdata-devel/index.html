Tiny generator of random data for Golang, also known as a faker

Randomdata is a tiny help suite for generating random data such as – first names (male or female) – last names – full names (male or female) – country names (full name or iso 3166.1 alpha-2 or alpha-3) – locales / language tags (bcp-47) – random email address – city names – American state names (two chars or full) – random numbers (in an interval) – random paragraphs – random bool values – postal- or zip-codes formatted for a range of different countries. – american sounding addresses / street names – silly names - suitable for names of things – random days – random months – random full date – random full profile – random date inside range – random phone number This package contains the source code needed for building packages that reference the following Go import paths: – github.com/Pallinder/go-randomdata
