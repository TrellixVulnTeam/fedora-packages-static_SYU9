perl-MIME-Lite-HTML

Provide routine to transform a HTML page in a MIME-Lite mail

  • Bodhi
  • Bugzilla
  • FAF
  • Koji
  • SCM

This module is a Perl mail client interface for sending message that support HTML format and build them for you.. This module provides routine to transform an HTML page in a MIME::Lite mail. So you need this module to use MIME-Lite-HTML possibilities.
