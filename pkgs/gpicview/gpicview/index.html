gpicview

Simple and fast Image Viewer for X

Gpicview is an simple and image viewer with a simple and intuitive interface. It's extremely lightweight and fast with low memory usage. This makes it very suitable as default image viewer of desktop system. Although it is developed as the primary image viewer of LXDE, the Lightweight X11 Desktop Environment, it only requires GTK+ and can be used in any desktop environment.
