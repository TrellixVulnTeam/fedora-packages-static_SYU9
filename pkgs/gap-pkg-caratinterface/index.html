This package provides GAP interface routines to some of the standalone programs of the package CARAT, developed by J. Opgenorth, W. Plesken, and T. Schulz at Lehrstuhl B f√ºr Mathematik, RWTH Aachen. CARAT is a package for computation with crystallographic groups. CARAT is to a large extent complementary to the GAP 4 package Cryst. In particular, it provides routines for the computation of normalizers and conjugators of finite unimodular groups in GL(n,Z), and routines for the computation of Bravais groups, which are all missing in Cryst. Furthermore, it provides a catalog of Bravais groups up to dimension 6. Cryst automatically loads Carat when it is available, and makes use of its functions where necessary. The Carat package thereby extends the functionality of the package Cryst considerably.
