perl-Games-Solitaire-Verify

Process and verify solitaire games

This is a CPAN Perl module that verifies the solutions of various variants of card Solitaire. It does not aim to try to be a solver for them, because this is too CPU intensive to be adequately done using perl5 (as of perl-5.10.0). If you're interested in the latter, look at: * https://fc-solve.shlomifish.org/ * https://fc-solve.shlomifish.org/links.html#other_solvers
