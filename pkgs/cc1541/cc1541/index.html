cc1541

Tool for creating Commodore Floppy disk images in D64, G64, D71 or D81 format

This is cc1541 v4.0, a tool for creating Commodore 1541 Floppy disk images in D64, G64, D71 or D81 format with custom sector interleaving etc. Also supports extended tracks 35-40 using either SPEED DOS or DOLPHIN DOS BAM-formatting.
