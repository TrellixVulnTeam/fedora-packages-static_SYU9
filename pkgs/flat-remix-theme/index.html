Flat Remix GTK theme is a pretty simple GTK window theme inspired on material design following a modern design using "flat" colors with high contrasts and sharp borders. Themes: - Flat Remix GTK - Flat Remix GTK Dark - Flat Remix GTK Darker - Flat Remix GTK Darkest Variants: - Solid: Theme without transparency - No Border: Darkest theme without white window border This meta package contains complete Flat Remix theme.
