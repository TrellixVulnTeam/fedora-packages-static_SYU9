Tuareg is an OCaml mode for GNU Emacs and XEmacs. It handles automatic indentation of Objective Caml and Caml Light code. Key parts of the code are highlighted using Font-Lock. Support to run an interactive Caml toplevel and debbuger is provided. Install this package if you need to edit OCaml code in XEmacs.
