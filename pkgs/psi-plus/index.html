psi-plus is the premiere Instant Messaging application designed for Microsoft Windows, Apple Mac OS X and GNU/Linux. Built upon an open protocol named Jabber, psi-plus is a fast and lightweight messaging client that utilises the best in open source technologies. psi-plus contains all the features necessary to chat, with no bloated extras that slow your computer down. The Jabber protocol provides gateways to other protocols as AIM, ICQ, MSN and Yahoo!.
