Native Rust encoder and decoder of TOML-formatted files and streams. Provides implementations of the standard Serialize/Deserialize traits for TOML data to facilitate deserializing and serializing Rust structures. This package contains library source intended for building other packages which use "the indexmap" feature of the "toml" crate.
