php-consolidation-log

Improved PSR-3 / Psr\\Log logger based on Symfony Console components

Consolidation\Log provides a Psr-3 compatible logger that provides styled log output to the standard error (stderr) stream. By default, styling is provided by the SymfonyStyle class from the Symfony Console component; however, alternative stylers may be provided if desired. Autoloader: /usr/share/php/Consolidation/Log/autoload.php
