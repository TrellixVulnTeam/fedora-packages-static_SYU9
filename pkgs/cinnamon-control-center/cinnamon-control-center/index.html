cinnamon-control-center

Utilities to configure the Cinnamon desktop

  • Builds
  • Updates
  • Bugs
  • Sources
  • Crash Reports
  • Koschei

This package contains configuration utilities for the Cinnamon desktop, which allow to configure accessibility options, desktop fonts, keyboard and mouse properties, sound setup, desktop theme and background, user interface properties, screen resolution, and other settings.
