kmid2

A MIDI/karaoke player for KDE

  • Bodhi
  • Bugzilla
  • FAF
  • Koji
  • SCM

KMid2 is a MIDI/karaoke file player, with configurable midi mapper, real Session Management, drag & drop, customizable fonts, etc. It has a very nice interface which let you easily follow the tune while changing the color of the lyrics. It supports output through external synthesizers, AWE, FM and GUS cards. It also has a keyboard view to see the notes played by each instrument.
