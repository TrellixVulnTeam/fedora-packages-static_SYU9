A python library to download and upload files from ANT-FS compliant devices (Garmin products).Any compliant ANT-FS device should in theory work, but those specific devices have been reported as working: Garmin Forerunner 60, Garmin Forerunner 405CX, Garmin Forerunner 310XT, Garmin Forerunner 610, Garmin Forerunner 910XT, Garmin FR70, Garmin Swim
