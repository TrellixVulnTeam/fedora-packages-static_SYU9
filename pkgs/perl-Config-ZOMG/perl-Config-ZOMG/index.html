Config::ZOMG is a fork of Config::JFDI. It removes a couple of unusual features and passes the same tests three times faster than Config::JFDI. Config::ZOMG is an implementation of Catalyst::Plugin::ConfigLoader. It will scan a directory for files matching a certain name. If such a file is found which also matches an extension that Config::Any can read, then the configuration from that file will be loaded.
