perl-Test-Some

Run a subset of tests

  • Bodhi
  • Bugzilla
  • FAF
  • Koji
  • SCM

This module allows one to run a subset of the 'subtest' tests given in a test file. The module declaration takes a whitelist of the subtests we want to run. Any subtest that doesn't match any of the whitelist items will be skipped (or potentially bypassed). The test files don't even need to be modified, as the module can also be invoked from the command-line.
