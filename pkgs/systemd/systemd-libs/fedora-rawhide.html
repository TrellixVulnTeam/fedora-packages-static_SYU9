2022-01-07 Zbigniew Jędrzejewski-Szmek <zbyszek at in dot waw dot pl> - 250.2-1 - Second stable release after v250: various bugfixes (systemd-resolved, systemd-journald, userdbctl, homed). - The manager should now gracefully handle the case where BPF LSM cannot be initialized (#2036145). The BPF filters are enabled again on all architectures, so *other* filter should also work on the affected architectures. - kernel-install now checks paths used by grub2 before sd-boot paths again (#2036199) - fstab-generator now ignores root-on-nfs/cifs/iscsi and live (#2037233) - CVE-2021-3997, #2024639: systemd-tmpfiles would exhaust the stack and crash during excessive recursion on a very deeply nested directory structure.
