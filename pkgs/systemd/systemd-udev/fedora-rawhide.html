Changelog

Date Author Change
2022-01-07 Zbigniew Jędrzejewski-Szmek <zbyszek at in dot waw dot pl> - 250.2-1 - Second stable release after v250: various bugfixes (systemd-resolved, systemd-journald, userdbctl, homed). - The manager should now gracefully handle the case where BPF LSM cannot be initialized (#2036145). The BPF filters are enabled again on all architectures, so *other* filter should also work on the affected architectures. - kernel-install now checks paths used by grub2 before sd-boot paths again (#2036199) - fstab-generator now ignores root-on-nfs/cifs/iscsi and live (#2037233) - CVE-2021-3997, #2024639: systemd-tmpfiles would exhaust the stack and crash during excessive recursion on a very deeply nested directory structure.
2022-01-04 Zbigniew Jędrzejewski-Szmek <zbyszek at in dot waw dot pl> - 250.1-1 - First stable version after v250: various bugfixes, in particular for sd-boot, systemd-networkd, and various build issues. - Fixes #2036517, #2035608, #2036217.
2021-12-30 Zbigniew Jędrzejewski-Szmek <zbyszek at in dot waw dot pl> - 250-3 - Disable bpf filters on arm64 (#2036145)
2021-12-25 Zbigniew Jędrzejewski-Szmek <zbyszek at in dot waw dot pl> - 250-2 - Fix warning about systemd-boot-update.service not existing on non-uefi architectures - Enable all bpf features (#2035608)
2021-12-23 Zbigniew Jędrzejewski-Szmek <zbyszek at in dot waw dot pl> - 250-1 - Version 250, only some very small changes since -rc3. - Switch unit status name format to 'combined' (#2028169)

2021-12-12 Zbigniew Jędrzejewski-Szmek <zbyszek at in dot waw dot pl> - 250~rc1-4 - Move systemd-boot-update.service to -udev subpackage and add it the the installation scriptlets (#2031400) - Move libcryptsetup-token-systemd plugins to -udev (#2031873) - Create /etc/resolv.conf symlink if nothing is present yet (#2032085)
