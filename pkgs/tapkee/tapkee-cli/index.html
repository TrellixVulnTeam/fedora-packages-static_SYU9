Tapkee is a cli-tool for efficient dimension reduction and provides implementations of the following dimension reduction methods: * Locally Linear Embedding and Kernel Locally Linear Embedding (LLE/KLLE) * Neighborhood Preserving Embedding (NPE) * Local Tangent Space Alignment (LTSA) * Linear Local Tangent Space Alignment (LLTSA) * Hessian Locally Linear Embedding (HLLE) * Laplacian eigenmaps * Locality Preserving Projections * Diffusion map * Isomap and landmark Isomap * Multidimensional scaling and landmark Multidimensional scaling (MDS/lMDS) * Stochastic Proximity Embedding (SPE) * PCA and randomized PCA * Kernel PCA (kPCA) * Random projection * Factor analysis * t-SNE * Barnes-Hut-SNE
