intel-mediasdk-tracer Subpackage of intel-mediasdk

Dump the calls of an application to the Intel Media SDK library

Media SDK Tracer is a tool which permits to dump logging information from the calls of the application to the Media SDK library. Trace log obtained from this tool is a recommended information to provide to Media SDK team on submitting questions and issues.
