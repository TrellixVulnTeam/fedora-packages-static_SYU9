mtpaint

Painting program for creating icons and pixel-based artwork

mtPaint is a simple painting program designed for creating icons and pixel-based artwork. It can edit indexed palette or 24 bit RGB images and offers basic painting and palette manipulation tools. Its main file format is PNG, although it can also handle JPEG, GIF, TIFF, BMP, XPM, and XBM files.
