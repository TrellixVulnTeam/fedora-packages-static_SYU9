Glade is a RAD tool to enable quick and easy development of user interfaces for the GTK+ 2 toolkit and the GNOME desktop environment. The user interfaces designed in Glade are saved as XML, which can be used in numerous programming languages including C, C++, Java, Perl, C#, Pike, Ruby, Haskell, Objective Caml and Scheme. Adding support for other languages is easy too. This package contains Glade for GTK+ 2.x. For GTK+ 3.x support, please install the 'glade' package instead.
