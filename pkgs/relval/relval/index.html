Relval can perform various tasks related to Fedora QA by interacting with the Fedora wiki. It lets you: * Create wiki pages for Fedora release validation test events * Generate statistics on release validation testing * Report release validation test results using a console interface See https://fedoraproject.org/wiki/QA/SOP_Release_Validation_Test_Event for more information on the process relval helps with.
