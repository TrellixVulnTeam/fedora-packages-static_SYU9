X2Go is a server-based computing environment with - session resuming - low bandwidth support - session brokerage support - client-side mass storage mounting support - client-side printing support - audio support - authentication by smartcard and USB stick This package provides an example configuration for providing the X2Go Plugin via an Apache webserver.
