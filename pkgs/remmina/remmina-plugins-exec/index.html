remmina-plugins-exec Subpackage of remmina

External execution plugin for Remmina Remote Desktop Client

Remmina is a remote desktop client written in GTK+, aiming to be useful for system administrators and travelers, who need to work with lots of remote computers in front of either large monitors or tiny net-books. This package contains the plugin to execute external processes (commands or applications) from the Remmina window.
