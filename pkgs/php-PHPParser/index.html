A PHP parser written in PHP to simplify static analysis and code manipulation. This package provides the library version 1. The php-nikic-php-parser3 package provides the library version 3. The php-nikic-php-parser4 package provides the library version 4. Autoloader: '/usr/share/php/PhpParser/autoload.php';
