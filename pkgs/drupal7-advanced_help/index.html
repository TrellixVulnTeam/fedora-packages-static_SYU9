The Advanced Help module provides a framework that allows module and theme developers to provide contextual in-line help within a Drupal site. It differs from the core Help module by separating help text from code, instead placing it in separate .html-files. These files can be edited by non-programmers, and may be included in the repo and distributed along with the module or theme project. This package provides the following Drupal modules: * advanced_help * help_example
