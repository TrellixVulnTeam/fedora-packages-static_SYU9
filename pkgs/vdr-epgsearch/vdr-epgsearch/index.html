vdr-epgsearch

Powerful schedules menu replacement plugin for VDR

  • Builds
  • Updates
  • Bugs
  • Sources
  • Crash Reports
  • Koschei

This plugin provides a powerful replacement for VDR's default schedules menu entry. It looks like the standard schedules menu, but adds several functions, such as additional commands for EPG entries, reusable queries which can be used as dynamic "search timers" etc.
