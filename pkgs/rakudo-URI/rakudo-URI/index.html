Perl6 realization of URI - Uniform Resource Identifiers handler A URI implementation using Perl 6 grammars to implement RFC 3986 BNF. Currently only implements parsing. Includes URI::Escape to (un?)escape characters that aren't otherwise allowed in a URI with % and a hex character numbering.
