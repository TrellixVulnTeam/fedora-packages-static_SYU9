After the Flood Sparks is a font that allows for the combination of text and visual data to show an idea and evidence in one headline. This builds on the principle of Sparklines defined by Edward Tufte and makes them easier to use. Sparklines are currently available as plugins or javascript elements. By installing the Sparks font you can use them immediately without the need for custom code. Sparks data needs to be formatted as comma-separated values, with curly brackets at both ends of the set, e.g., {30,60,90}. This package provides the Dot-line family.
