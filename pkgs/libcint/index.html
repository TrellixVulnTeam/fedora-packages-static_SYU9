libcint

General Gaussian-type orbitals integrals for quantum chemistry

libcint is an open source library for analytical Gaussian integrals. It provides C/Fortran API to evaluate one-electron / two-electron integrals for Cartesian / real-spherical / spinor Gaussian type functions.
