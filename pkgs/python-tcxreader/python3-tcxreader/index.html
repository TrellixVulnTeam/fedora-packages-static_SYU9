tcxreader is a parser/reader for Garmin's TCX file format

This is a simple TCX parser / reader which can read Garmin TCX file extension files. The following data is currently parsed: longitude, latitude, elevation, time, distance, hr_value, cadence, watts, TPX_speed (extension). It also works well with missing data!
