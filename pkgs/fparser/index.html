fparser

Function parser library for C++

  • Bodhi
  • Bugzilla
  • FAF
  • Koji

This C++ library offers a class which can be used to parse and evaluate a mathematical function from a string (which might be for example requested from the user). The syntax of the function string is similar to mathematical expressions written in C/C++.
