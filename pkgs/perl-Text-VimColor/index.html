perl-Text-VimColor

Syntax color text in HTML or XML using Vim

  • Bodhi
  • Bugzilla
  • FAF
  • Koji
  • SCM

This module tries to markup text files according to their syntax. It can be used to produce web pages with pretty-printed colorful source code samples. It can produce output in various formats. text-vimcolor is a command line interface to the Perl module Text::VimColor: text-vimcolor --format html --filetype prolog foo.pl > foo.html
