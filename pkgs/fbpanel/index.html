fbpanel

A lightweight X11 desktop panel

  • Bodhi
  • Bugzilla
  • FAF
  • Koji
  • SCM

fbpanel is a lightweight X11 desktop panel. It works with any ICCCM / NETWM compliant window manager such as sawfish, metacity, openbox, xfwm4, or KDE. It features tasklist, pager, launchbar, clock, menu and systray.
