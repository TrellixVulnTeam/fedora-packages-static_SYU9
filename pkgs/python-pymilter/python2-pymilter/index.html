python2-pymilter Subpackage of python-pymilter

Python interface to sendmail milter API

This is a python extension module to enable python scripts to attach to sendmail's libmilter functionality. Additional python modules provide for navigating and modifying MIME parts, sending DSNs, and doing CBV.
