perl-Pod-Simple-Wiki

Utility and perl classes for converting POD to Wiki text

  • Builds

  • Koschei

The Pod::Simple::Wiki module is used for converting Pod text to Wiki text. It currently contains the following output filters: Confluence, Kwiki, Mediawiki, Moinmoin, Template, Tiddlywiki, Twiki and Usemod.
