ogre-samples Subpackage of ogre

Ogre samples executables and media

  • Builds

This package contains the compiled (not the source) sample applications coming with Ogre. It also contains some media (meshes, textures,...) needed by these samples. The samples are installed in /usr/lib64/Samples/*.so and can be run using SampleBrowser.
