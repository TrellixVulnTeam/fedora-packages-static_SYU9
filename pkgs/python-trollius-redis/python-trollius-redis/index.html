Redis client for Python trollius This Redis library is a completely asynchronous, non-blocking client for a Redis server. It depends on trollius (asyncio compatible for PEP 3156). It supports Python 2 Trollius-using developers. If you're new to asyncio, it can be helpful to check out the asyncio documentation first.
