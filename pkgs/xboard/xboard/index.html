xboard

An X Window System graphical chessboard

  • Builds
  • Updates

Xboard is an X Window System based graphical chessboard which can be used with the GNU chess and Crafty chess programs, with Internet Chess Servers (ICSs), with chess via email, or with your own saved games. Install the xboard package if you need a graphical chessboard.
