Debug::ShowStuff grew dynamically from my needs in debugging code. I found myself doing the same tasks over and over... displaying the keys and values in a hash, displaying the elements in an array, displaying the output of STDERR in a web page, etc. Debug::ShowStuff began as two or three of my favorite routines and grew as I added to that collection. Finally I decided to publish these tools in the hope that other Perl programmers will find them useful.
