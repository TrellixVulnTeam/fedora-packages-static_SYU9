JS-Yaml JavaScript library packaged for setup-tools (easy_install) / pip. This package is intended to be used by any project that needs these files. It intentionally does not provide any extra code except some metadata nor has any extra requirements. This package provides Python 3 build of XStatic-JS-Yaml.
