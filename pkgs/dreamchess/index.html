DreamChess is an open source chess game. Features: - 3D OpenGL graphics - various chess board sets: from classic wooden to flat figurines - music, sound effects - on-screen move lists using SAN notation - undo functionality - save-games in PGN format A moderately strong chess engine as a sub-package: Dreamer.
