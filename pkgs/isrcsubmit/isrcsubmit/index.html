isrcsubmit

Script to submit ISRCs from disc to MusicBrainz

  • Builds
  • Updates
  • Bugs
  • Sources
  • Crash Reports
  • Koschei

This python script extracts ISRCs from audio cds and submits them to MusicBrainz. Features: read and submit ISRCs from disc, search for releases with the TOC of the disc, submit discIds / TOCs, display release information from MusicBrainz, duplicate ISRC detection (local and on server), keyring support for login information.
