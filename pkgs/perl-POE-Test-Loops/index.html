perl-POE-Test-Loops

Reusable tests for POE::Loop authors

  • Bodhi

  • Koji

POE::Test::Loops contains one function, generate(), which will generate all the loop tests for one or more POE::Loop subclasses. The poe-gen-tests manual page also documents the POE::Test::Loops system in more detail.
