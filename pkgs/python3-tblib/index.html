Traceback serialization library that allows you to: * Pickle tracebacks and raise exceptions with pickled tracebacks in different processes. This allows better error handling when running code over multiple processes (imagine multiprocessing, billiard, futures, celery, etc). * Parse traceback strings and raise with the parsed tracebacks.
