python-wikitcms is a library for interacting with Fedora's wiki-based 'test management' system. It can: * Create the pages for release validation test events * Find existing release validation event pages, in various ways * Report test results The wiki-based test management system itself is documented at: https://fedoraproject.org/wiki/Wikitcms This is the Python 2 build.
