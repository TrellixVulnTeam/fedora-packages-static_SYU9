NetPyNE is a Python package to facilitate the development, simulation, parallelization, analysis, and optimization of biological neuronal networks using the NEURON simulator. For more details, installation instructions, documentation, tutorials, forums, videos and more, please visit: www.netpyne.org This package is developed and maintained by the Neurosim lab (www.neurosimlab.org)
