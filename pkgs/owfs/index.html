Subpackages

  • owfs-capi - C-API to develop third-part applications which access 1-Wire networks
  • owfs-devel - Files for development of OWFS applications
  • owfs-fs - Virtual file-system on top of owfs-libs providing access to 1-Wire networks
  • owfs-ftpd - FTP daemon providing access to 1-Wire networks
  • owfs-httpd - HTTP daemon providing access to 1-Wire networks
  • owfs-libs - Core library providing base functions to other OWFS modules
  • owfs-mon - Statistics and settings monitor for owserver
  • owfs-ownet - C-API to develop third-part applications which access 1-Wire networks
  • owfs-php - PHP interface for the 1-wire file-system
  • owfs-server - Back-end server (daemon) for 1-wire control
  • owfs-shell - Light weight shell access to owserver and the 1-wire file-system
  • owfs-tap - Packet sniffer for the owserver protocol
  • owfs-tcl - Tcl interface for the 1-wire file-system
