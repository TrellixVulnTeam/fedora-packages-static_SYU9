textern

Firefox add-on for editing text in your favorite external editor

Textern is a Firefox add-on that allows you to edit text areas in web pages using an external editor. It is similar in functionality to the popular It's All Text! add-on, though makes use of the WebExtension API and is thus fully compatible with multiprocessing and supported beyond Firefox 57. This is not a self-standing Firefox add-on, it's only the "native" application used by Add-on named "textern". Please install the Add-on manually.
