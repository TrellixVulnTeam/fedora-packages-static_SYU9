Changelog

Date Author Change
2014-06-06 Björn Persson <bjorn at rombobjörn dot se> - 3.8.2-1 - Upgraded to 3.8.2. - Enabled the OpenGL bindings. - Excluded the GPS plug-in.
2014-05-10 Björn Persson <bjorn at rombobjörn dot se> - 3.4.2-3 - Build the demo programs in the check phase instead of the build phase.
2014-02-04 Björn Persson <bjorn at rombobjörn dot se> - 3.4.2-2 - Verify that the generated Ada code is the same as what the developers upstream have reviewed and tested.
2014-01-28 Björn Persson <bjorn at rombobjörn dot se> - 3.4.2-1 - New package for GTKada 3.x, partly based on the existing package GtkAda.
