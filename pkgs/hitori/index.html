hitori

Logic puzzle game for GNOME

A small application written to allow one to play the Hitori puzzle game, which is similar in theme to more popular puzzles such as Sudoku. It has full support for playing the game (i.e. it checks all three rules are satisfied). It has undo/redo support, can give hints, and allows for cells to be tagged with one of two different tags, to aid in solving the puzzle. It has support for anything from 5×5 to 10×10 grids.
