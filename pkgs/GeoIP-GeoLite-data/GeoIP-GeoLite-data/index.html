GeoIP-GeoLite-data

Free GeoLite IP geolocation country database

The GeoLite databases are free IP geolocation databases. This package contains a database that maps IPv4 addresses to countries. This product includes GeoLite data created by MaxMind, available from http://www.maxmind.com/
