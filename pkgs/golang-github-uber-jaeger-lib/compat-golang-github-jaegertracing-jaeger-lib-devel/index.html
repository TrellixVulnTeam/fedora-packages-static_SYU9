A collection of shared infrastructure libraries used by different components of Jaeger backend and jaeger-client-go. This library is not intended to be used standalone, and provides no guarantees of backwards compatibility. If your Go code depends on this package, you should patch it to import directly github.com/uber/jaeger-lib.
