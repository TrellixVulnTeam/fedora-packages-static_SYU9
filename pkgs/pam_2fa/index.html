pam_2fa

Second factor authentication for PAM

  • Bodhi
  • Bugzilla
  • FAF
  • Koji
  • SCM

The PAM 2FA module provides a second factor authentication, which can be combined with the standard PAM-based password authentication to ask for: * What you know: user account password ( standard PAM modules ) * What you have (pick one of): (PAM 2FA) * A Google Authenticator Application on your phone * A Phone Number capable of receiving SMS * A Yubikey
