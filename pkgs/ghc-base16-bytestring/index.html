ghc-base16-bytestring

Fast hex encoding and decoding for ByteStrings

  • Bodhi
  • Bugzilla
  • FAF
  • Koji
  • SCM

This package provides support for encoding and decoding binary data according to 'base16' (see also ) for strict (see "Data.ByteString.Base16") and lazy 'ByteString's (see "Data.ByteString.Base16.Lazy"). See also the package which provides an uniform API providing conversion paths between more binary and textual types.
