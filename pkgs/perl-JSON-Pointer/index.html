perl-JSON-Pointer

Perl implementation of JSON Pointer

  • Bodhi

This library is implemented JSON Pointer draft version 9 and some useful operators from JSON Patch draft version 10 . JSON Pointer is available to identify a specified value, and it is similar to XPath. Please see the both of specifications for details.
