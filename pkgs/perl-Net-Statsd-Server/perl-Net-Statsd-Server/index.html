perl-Net-Statsd-Server

Library for the Perl port of Flickr/Etsy's statsd metrics daemon

Net::Statsd::Server is the server component of statsd. It implements a daemon that listens on a given host/port for incoming UDP packets and dispatches them to whatever you want, including Graphite or your console. Look into the Net::Statsd::Server::Backend::* name space to know all the possibilities, or write a back-end yourself.
