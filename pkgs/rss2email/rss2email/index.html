rss2email

Deliver news from RSS feeds to your SMTP server as text or HTML mail

  • Builds

  • Koschei

rss2email lets you subscribe to a list of XML news feeds (RSS or Atom). It can parse them regularly with the help of cron and send new items to you by email. An HTML mail will be send in the default configuration to the local SMTP server. See the manual page r2e for details on how to set up rss2email.
