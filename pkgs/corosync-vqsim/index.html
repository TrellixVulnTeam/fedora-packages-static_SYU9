corosync-vqsim Subpackage of corosync

The Corosync Cluster Engine - Votequorum Simulator

  • Bodhi

A command-line simulator for the corosync votequorum subsystem. It uses the same code as the corosync quorum system but forks them into subprocesses to simulate nodes. Nodes can be added and removed as well as partitioned (to simulate network splits)
