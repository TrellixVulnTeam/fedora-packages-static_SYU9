pngnq

Pngnq is a tool for quantizing PNG images in RGBA format

Pngnq is a tool for quantizing PNG images in RGBA format. The neuquant algorithm uses a neural network to optimise the color map selection. This is fast and quite accurate, giving good results on many types of images.
