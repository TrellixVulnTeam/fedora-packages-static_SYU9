The \boolexpr macro evaluates boolean expressions in a purely expandable way. \boolexpr{ A \OR B \AND C } expands to 0 if the logical expression is TRUE. A, B, C may be: numeric expressions such as: x=y, x<>y, x>y or x} ... \case{} ... ... \otherwise ... \endswitch
