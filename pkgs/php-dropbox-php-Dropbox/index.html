This PHP library allows you to easily integrate dropbox with PHP. The library makes use of OAuth. Optional Dependencies: Zend framework: Zend_{Oauth,Json,Uri} for OAuth/Zend back-end Wordpress: WP_Http for OAuth/Wordpress back-end
