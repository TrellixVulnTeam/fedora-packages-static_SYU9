factory_boy is a fixtures replacement based on thoughtbot's factory_girl . Its features include: - Straightforward syntax - Support for multiple build strategies (saved/unsaved instances, attribute dicts, stubbed objects) - Powerful helpers for common cases (sequences, sub-factories, reverse dependencies, circular factories, ...) - Multiple factories per class support, including inheritance - Support for various ORMs (currently Django, Mogo, SQLAlchemy)
