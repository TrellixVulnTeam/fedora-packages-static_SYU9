drupal7-domain_locale

Adds language handling functionality per domain basis

  • Bodhi
  • Bugzilla
  • FAF
  • Koji
  • SCM

Domain Locale allows to customize language sets per domain for Drupal installs using Domain Access and core Locale module. This module also offers drush integration. You need this module if: * you want a different default language on different domains * you want a different language set for each different domain This package provides the following Drupal module: * domain_locale
