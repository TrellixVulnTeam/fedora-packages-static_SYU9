liberation-serif-fonts Subpackage of liberation-fonts

Serif fonts to replace commonly used Microsoft Times New Roman

The Liberation Fonts are intended to be replacements for the 3 most commonly used fonts on Microsoft systems: Times New Roman, Arial, and Courier New. This package provides Serif TrueType fonts that replace commonly used Microsoft Times New Roman.
