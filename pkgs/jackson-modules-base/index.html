Subpackages

  • jackson-modules-base - Jackson modules: Base
  • jackson-module-afterburner - Jackson module that uses byte-code generation to further speed up data binding

  • jackson-module-jaxb-annotations - Support for using JAXB annotations as an alternative to "native" Jackson annotations
  • jackson-module-mrbean - Functionality for implementing interfaces and abstract types dynamically
  • jackson-module-osgi - Jackson module to inject OSGI services in deserialized beans
  • jackson-module-paranamer - Jackson module that uses Paranamer to introspect names of constructor params
  • jackson-modules-base-javadoc - Javadoc for jackson-modules-base
