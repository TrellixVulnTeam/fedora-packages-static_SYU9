ipxripd

IPX RIP/SAP daemon - routing for IPX networks

  • Bodhi
  • Bugzilla
  • FAF
  • Koji
  • SCM

ipxripd is an implementation of Novell's RIP and SAP protocols. It automagically builds and updates IPX routing table in the Linux kernel. ipxripd can be useful to get a Linux box to act as an IPX router.
