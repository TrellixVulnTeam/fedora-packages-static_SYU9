coq-menhirlib Subpackage of ocaml-menhir

Support library for verified Coq parsers produced by Menhir

  • Bodhi
  • Bugzilla
  • FAF
  • Koji
  • SCM

The Menhir parser generator, in --coq mode, can produce Coq parsers. These parsers must be linked against this library, which provides both an interpreter (which allows running the generated parser) and a validator (which allows verifying, at parser construction time, that the generated parser is correct and complete with respect to the grammar).
