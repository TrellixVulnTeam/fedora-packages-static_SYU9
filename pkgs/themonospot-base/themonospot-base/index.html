themonospot-base

Base component for themonospot system, parser/editor and content descriptor

themonospot-base is core package for themonospot system. It install: - themonospot-base mono assembly (use from other gui applications) - themonospot-plugin-interface (use to write plugins) - themonospot-plugin-manager (use to load plugins at runtime) - xml language files
