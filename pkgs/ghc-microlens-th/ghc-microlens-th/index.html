ghc-microlens-th

Automatic generation of record lenses for microlens

  • Builds
  • Updates
  • Bugs
  • Sources
  • Crash Reports
  • Koschei

This package lets you automatically generate lenses for data types; code was extracted from the lens package, and therefore generated lenses are fully compatible with ones generated by lens (and can be used both from lens and microlens). This package is a part of the <http://hackage.haskell.org/package/microlens microlens> family; see the readme <https://github.com/monadfix/microlens#readme on Github>.
