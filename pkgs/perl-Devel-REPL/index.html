perl-Devel-REPL

Modern perl interactive shell

This is an interactive shell for Perl, commonly known as a REPL - Read, Evaluate, Print, Loop. The shell provides for rapid development or testing of code without the need to create a temporary source code file. Through a plugin system, many features are available on demand. These plugins are available: Completion CompletionDriver::INC CompletionDriver::Keywords DDC DDS Interrupt LexEnv MultiLine::PPI Nopaste PPI Refresh The plugins are available in standalone RPM packages. For example the MultiLine::PPI plugin is delivered within perl-Devel-REPL-MultiLine-PPI package.
