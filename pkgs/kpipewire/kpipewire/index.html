kpipewire

Set of convenient classes to use PipeWire in Qt projects

  • Builds

  • Sources

  • Koschei

It is developed in C++ and it's main use target is QML components. As it's what's been useful, this framework focuses on graphical PipeWire features. If it was necessary, these could be included. At the moment we offer two main components: - KPipeWire: offers the main components to connect to and render PipeWire into your app. - KPipeWireRecord: using FFmpeg, helps to record a PipeWire video stream into a file.
