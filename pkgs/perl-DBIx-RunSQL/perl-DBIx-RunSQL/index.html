perl-DBIx-RunSQL

Run SQL commands from a file

  • Builds
  • Updates
  • Bugs
  • Sources

  • Koschei

This module abstracts away the "run these SQL statements to set up a database" into a module. It also abstracts away the reading of SQL from a file and allows for various command line parameters to be passed in.
