perl-rpm-build-perl

Perl compiler back-end to extract Perl dependencies

  • Builds
  • Updates
  • Bugs
  • Sources
  • Crash Reports
  • Koschei

B::PerlReq is a back-end module for the Perl compiler that extracts dependencies from Perl source code, based on the internal compiled structure that Perl itself creates after parsing a program. The output of B::PerlReq is suitable for automatic dependency tracking (e.g. for RPM packaging).
