TiLP2 is a Texas Instruments handhelds <-> PC communication program for Linux. It is able to use any type of link cable (Gray/Black/Silver/Direct Link) with any calculator. See http://lpg.ticalc.org/. With TiLP, you can transfer files from your PC to your Texas Instruments calculator, and vice-versa. You can also make a screen dump, send/receive data, backup/restore contents, install FLASH applications, or upgrade the OS.
