Hypervvssd is an implementation of Hyper-V VSS functionality for Linux. The daemon is used for host initiated guest snapshot on Hyper-V hypervisor. The daemon first registers with the kernel driver. After this is done it waits for instructions from Windows Host if to "freeze" or "thaw" the filesystem on the Linux Guest.
