Hyper-V key value pair (KVP) daemon

Hypervkvpd is an implementation of Hyper-V key value pair (KVP) functionality for Linux. The daemon first registers with the kernel driver. After this is done it collects information requested by Windows Host about the Linux Guest. It also supports IP injection functionality on the Guest.
