PyVO is a package providing access to remote data and services of the Virtual Observatory (VO) using Python. The pyvo module currently provides these main capabilities: * Find archives that provide particular data of a particular type and/or relates to a particular topic * Search an archive for datasets of a particular type * Do simple searches on catalogs or databases * Get information about an object via its name
