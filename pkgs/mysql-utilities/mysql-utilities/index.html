A package of utilities that are used for maintenance and administration of MySQL servers. These utilities encapsulate a set of primitive commands, and bundles them so they can be used to perform macro operations with a single command. Documentation: http://dev.mysql.com/doc/workbench/en/mysql-utilities.html
