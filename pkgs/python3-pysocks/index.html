A fork of SocksiPy with bug fixes and extra features. Acts as a drop-in replacement to the socket module. Featuring: - SOCKS proxy client for Python 2.6 - 3.x - TCP and UDP both supported - HTTP proxy client included but not supported or recommended (you should use urllib2's or requests' own HTTP proxy interface) - urllib2 handler included. This package is for Python3 version 3.9 only.
