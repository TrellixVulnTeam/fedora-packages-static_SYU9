Provides an API to ensure that the PECL Mongo extension can be used consistently across various Horde packages. Tu use this module, you also need to install - php-pecl-mongo (PHP 5 only) or - php-pecl-mongodb (PHP 5 or 7) - php-alcaeus-mongo-php-adapter
