perl-Pod-Plainer

Perl extension for converting modern POD to old-style POD

  • Builds
  • Updates
  • Bugs
  • Sources
  • Crash Reports
  • Koschei

This Perl module takes POD with the new 'C<< .. >>' constructs and returns the older style with just 'C<>'. '<' and '>' are replaced by 'E<lt>' and 'E<gt>'. This can be used to preprocess POD before using tools which do not recognize the new style PODs.
