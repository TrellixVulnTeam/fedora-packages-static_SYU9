lua-xmlrpc

Lua package to access and provide XML-RPC services

Lua XML-RPC is a library to make remote procedure calls using XML-RPC. It also offers facilities to develop server-side software. Lua XML-RPC provides a simple API and an abstraction layer over XML, thus avoiding the complexity of having to deal with strings that represent data structures.
