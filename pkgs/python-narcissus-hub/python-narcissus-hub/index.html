This package contains the "hub" components for narcissus. Narcissus is a web application that visualizes web server hits as they happen in real time. Features: * IP addresses converted to latitude/longitude, then streamed via WebSockets to `polymaps <http://polymaps.org/>`_. * Realtime graphs of what countries are downloading what content with `d3 <http://d3js.org>`_. * `Ã˜mq (zeromq) <http://www.zeromq.org/>`_ on the backend. * **Fast**. No polling. You can see a demo running live at http://narcissus.rc.rit.edu It is visualizing the logs of http://mirror.rit.edu
