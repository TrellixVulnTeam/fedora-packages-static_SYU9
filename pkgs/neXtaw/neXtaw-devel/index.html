neXtaw is a replacement library for the Athena (libXaw) widget set. It is based on Xaw3d, by Kaleb Keithley and is almost 100% backward compatible with it. Its goal is to try to emulate the look and feel of the N*XTSTEP GUI. This package contains the development files of the neXtaw library.
