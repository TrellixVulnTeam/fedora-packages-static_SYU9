mediawiki-wikicalendar

Simple calendar extension for mediawiki

  • Builds
  • Updates
  • Bugs
  • Sources
  • Crash Reports
  • Koschei

The extension adds an <calendar> tag to the mediawiki syntax which can show calendars in different formats. The idea was to keep the calendar as simple as possible and leave all the editing and page creation to mediawiki. It doesn't require any new database tables or files.
