(#1885101, #1890632, #1879216) A backwards-incompatible change affects PCI network devices which are connected through a bridge which is itself associated with a slot. When more than one device was associated with the same slot, one of the devices would pseudo-randomly get named after the slot. That name is now not generated at all. This changed behaviour is causes the net naming scheme to be changed to "v247".
