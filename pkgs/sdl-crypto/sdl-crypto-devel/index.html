Crypto provides a simple implementation of the Rijndael (now AES) and DES encryption algorithms as well as the SHA1 and SHA2 hashing algorithms. The library is almost a verbatim copy of the code from OpenBSD and PuTTY for use by the SDL-Hercules-390 emulator. The sdl-crypto-devel package contains libraries and header files for developing applications that use sdl-crypto.
