statgrab Subpackage of libstatgrab

Sysctl-style interface to the statistics from libstatgrab

Statgrab gives a sysctl-style interface to the statistics gathered by libstatgrab. This extends the use of libstatgrab to people writing scripts or anything else that can't easily make C function calls. Included with statgrab is a script to generate an MRTG configuration file to use statgrab.
