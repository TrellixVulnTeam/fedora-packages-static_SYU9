The CM-LGC PostScript Type 1 fonts are converted from the METAFONT sources of the Computer Modern font families. CM-LGC supports the T1, T2A, LGR, and TS1 encodings, i.e. Latin, Cyrillic, and Greek. This package consists of files used by other ctan-cm-lgc-fonts packages.
