arc-kde

Port of the popular GTK theme Arc for the Plasma 5 desktop

This is a port of the popular GTK theme Arc for Plasma 5 desktop with a few additions and extras. In this package you'll find: - Aurorae Themes - Konsole Color Schemes - Konversation Themes - Kvantum Themes - Plasma Color Schemes - Plasma Desktop Themes - Plasma Look-and-Feel Settings - Wallpapers - Yakuake Skins - Extra tools
