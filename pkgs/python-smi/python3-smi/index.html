PySMI is a pure-Python implementation of SNMP SMI MIB parser. This tool is designed to turn ASN.1 MIBs into various formats. As of this moment, JSON and pysnmp modules can be generated from ASN.1 MIBs. - Understands SMIv1, SMIv2 and de-facto SMI dialects - Turns MIBs into pysnmp classes and JSON documents - Maintains an index of MIB objects over many MIB modules - Automatically pulls ASN.1 MIBs from local directories, ZIP archives, HTTP and FTP servers
