R-jpeg

Read and write JPEG images

  • Builds
  • Updates
  • Bugs
  • Sources

  • Koschei

This package provides an easy and simple way to read, write and display bitmap images stored in the JPEG format. It can read and write both files and in-memory raw vectors.
