perl-Math-ConvexHull-MonotoneChain

Monotone chain algorithm for finding a convex hull in 2D

  • Bodhi
  • Bugzilla
  • FAF
  • Koji
  • SCM

This is somewhat experimental still. This (XS) module optionally exports a single function C which calculates the convex hull of the input points and returns it. The algorithm is C due to having to sort the input list, but should be somewhat faster than a plain Graham's scan (also C) in practice since it avoids polar coordinates.
