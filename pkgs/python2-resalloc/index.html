python2-resalloc Subpackage of resalloc

Resource allocator for expensive resources - Python 2 client library

The resalloc project aims to help with taking care of dynamically allocated resources, for example ephemeral virtual machines used for the purposes of CI/CD tasks. The python2-resalloc package provides Python 2 client library for talking to the resalloc server.
