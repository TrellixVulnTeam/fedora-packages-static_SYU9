DBIx::XHTML_Table is a DBI extension that creates an XHTML table from a database query result set. It was created to fill the gap between fetching rows from a database and transforming them into a web browser renderable table. DBIx::XHTML_Table is intended for programmers who want the responsibility of presenting (decorating) data, easily. This module is meant to be used in situations where the concern for presentation and logic separation is overkill. Providing logic or editable data is beyond the scope of this module, but it is capable of doing such.
