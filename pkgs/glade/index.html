Glade is a RAD tool to enable quick and easy development of user interfaces for the GTK+ toolkit and the GNOME desktop environment. The user interfaces designed in Glade are saved as XML, which can be used in numerous programming languages including C, C++, C#, Vala, Java, Perl, Python, and others.
