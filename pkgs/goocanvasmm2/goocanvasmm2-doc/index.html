This package contains developer's documentation for the goocanvasmm2 library. Goocanvasmm2 is the C++ API for the goocanvas graphics library. The documentation can be viewed either through the devhelp documentation browser or through a web browser. If using a web browser the documentation is at /usr/share/doc/goocanvasmm-2.0
