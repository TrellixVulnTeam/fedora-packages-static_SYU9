mt32emu

C/C++ library for emulating Roland MT-32, CM-32L and LAPC-I synthesizer modules

mt32emu is a part of the Munt project. It represents a C/C++ library named libmt32emu which allows to emulate (approximately) the Roland MT-32, CM-32L and LAPC-I synthesizer modules.
