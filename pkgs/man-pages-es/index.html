This package contains the translation into Spanish of the English man-pages package. It is a beta release, so you can still find a lot of bugs. Contributions are welcome. For any doubt or suggestion about this release, send an e-mail to Juan Piernas Canovas . In order to collaborate with the project, please visit http://es.tldp.org.
