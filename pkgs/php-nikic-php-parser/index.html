This is a PHP 5.2 to PHP 7.0 parser written in PHP. Its purpose is to simplify static code analysis and manipulation. This package provides the library version 2. The php-PHPParser package provides the library version 1 and the php-parse command. Documentation: https://github.com/nikic/PHP-Parser/tree/master/doc Autoloader: /usr/share/php/PhpParser2/autoload.php
