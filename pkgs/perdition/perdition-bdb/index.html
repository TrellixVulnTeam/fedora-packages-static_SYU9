perdition-bdb Subpackage of perdition

Library to allow perdition to access Berkely DB based pop maps

  • Builds
  • Updates
  • Bugs
  • Sources

Perdition allows for arbitrary user database access through shared libraries much in the manner of NSS in glibc. This package allows a user database to be sourced from a Berkely DB.
