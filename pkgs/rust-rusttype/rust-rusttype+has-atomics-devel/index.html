Pure Rust alternative to libraries like FreeType. RustType provides an API for loading, querying and rasterising TrueType fonts. It also provides an implementation of a dynamic GPU glyph cache for hardware font rendering. This package contains library source intended for building other packages which use "has-atomics" feature of "rusttype" crate.
