This repository contains the NeuroML 2 Schema, the ComponentType definitions in LEMS and a number of example files in NeuroML 2 and LEMS files for running simulations. For more details on LEMS and NeuroML 2 see: Robert C. Cannon, Padraig Gleeson, Sharon Crook, Gautham Ganapathy, Boris Marin, Eugenio Piasini and R. Angus Silver, LEMS: A language for expressing complex biological models in concise and hierarchical form and its use in underpinning NeuroML 2, Frontiers in Neuroinformatics 2014, doi:10.3389/fninf.2014.00079
