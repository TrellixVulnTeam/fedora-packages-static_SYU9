This Perl module contains the support routines for parsing XML and generating the TrueType font structures as a result. The module has been separated from the rest of the perl-Font-TTF package in order to reduce the dependency that this would bring, of the whole package on XML::Parser. This way, people without the XML::Parser can still use the rest of the package.
