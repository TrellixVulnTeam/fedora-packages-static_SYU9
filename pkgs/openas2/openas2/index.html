OpenAS2 is a java-based implementation of the EDIINT AS2 standard. It is intended to be used as a server. It is extremely configurable and supports a wide variety of signing and encryption algorithms. This package has a mostly preconfigured openas2 server, using hsqldb by default for a standalone package. It can be configured to use postgresql instead.
