nagios-plugins-bonding

Nagios plugin to monitor Linux bonding interfaces

  • Builds

  • Koschei

This package contains check_linux_bonding, which is a plugin for Nagios that checks bonding network interfaces on Linux. The plugin will report any interfaces that are down (both masters and slaves), as well as other aspects which may point to a problem with bonded interfaces.
