perl-Snowball-Norwegian

Porter's stemming algorithm for Norwegian

The stem function takes a scalar as a parameter and stems the word according to Martin Porter's Norwegian stemming algorithm, which can be found at the Snowball website: http://snowball.tartarus.org/.
