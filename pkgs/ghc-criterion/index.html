ghc-criterion

Robust, reliable performance measurement and analysis

This library provides a powerful but simple way to measure software performance. It provides both a framework for executing and analysing benchmarks and a set of driver functions that makes it easy to build and run benchmarks, and to analyse their results. The fastest way to get started is to read the , followed by the documentation and examples in the "Criterion.Main" module. For examples of the kinds of reports that criterion generates, see .
