perl-Dist-Zilla-Plugin-CopyFilesFromRelease

Copy files from a release for SCM inclusion

This Dist::Zilla plugin will automatically copy the files that you specify in dist.ini from the build directory into the distribution directory. This is so you can commit them to version control.
