python3-jpype Subpackage of jpype

Full access for Python programs to Java class libraries

JPype is an effort to allow python programs full access to java class libraries. This is achieved not through re-implementing Python, as Jython/JPython has done, but rather through interfacing at the native level in both Virtual Machines. This is the package with support for Python version 3.9.
