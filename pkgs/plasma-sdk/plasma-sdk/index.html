Plasma SDK contains the following tools for Plasma-related development: - CuttleFish - icon theme browser - EngineExplorer - tool to browse and interact with data engines - PlasmoidViewer - an isolated Plasma environment for testing applets - ThemeExplorer - shows all components of a widget theme
