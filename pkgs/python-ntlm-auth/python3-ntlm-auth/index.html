python3-ntlm-auth Subpackage of python-ntlm-auth

Python 3 compatible NTLM library (requires md4, thus legacy OpenSSL settings)

  • Builds
  • Updates
  • Bugs
  • Sources

  • Koschei

This package allows Python clients running on any operating system to provide NTLM authentication to a supporting server. With OpenSSL 3 or above, the legacy OpenSSL provider needs to be set in order to support md4 in Python.
