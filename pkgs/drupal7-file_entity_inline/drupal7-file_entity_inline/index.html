drupal7-file_entity_inline

Makes field entities editable within other entities

  • Koschei

This module aims to provide the ability to edit File entities [1] inline, as part of another form (such as nodes). This should enable the ability for reusable and site-wide editing of file properties, such as description, alt text, captions, bylines, etc. This package provides the following Drupal module: * file_entity_inline [1] http://drupal.org/project/file_entity
