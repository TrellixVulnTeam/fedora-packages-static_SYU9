perl-Gtk3-SimpleList

Simple interface to Gtk3's complex MVC list widget

  • Builds
  • Updates

  • Koschei

Gtk3 has a powerful, but complex MVC (Model, View, Controller) system used to implement list and tree widgets. Gtk3::SimpleList Perl module automates the complex setup work and allows you to treat the list model as a more natural list of lists structure.
