nvmetcli

An adminstration shell for NVMe storage targets

  • Builds
  • Updates
  • Bugs
  • Sources
  • Crash Reports
  • Koschei

This package contains the command line interface to the NVMe over Fabrics nvmet in the Linux kernel. It allows configuring the nvmet interactively as well as saving / restoring the configuration to / from a json file.
