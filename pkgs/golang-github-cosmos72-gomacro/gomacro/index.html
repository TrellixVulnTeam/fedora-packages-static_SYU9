Go interpreter and debugger with REPL, Eval, generics and Lisp-like macros

Gomacro is an almost complete Go interpreter, implemented in pure Go. It offers both an interactive REPL and a scripting mode, and does not require a Go toolchain at runtime (except in one very specific case: import of a third party package at runtime).
