More flexible version of `scoped-tls`, allowing the following additional features: - Storage of references to dynamically sized types. - Storage of mutable references. - Storage of types containing unbound lifetime parameters (higher- kinded types). - Some combination of the above. This package contains library source intended for building other packages which use "scoped-tls-hkt" crate.
