Fedora Finder finds Fedora. For now, that means it finds Fedora images - for stable releases, milestone pre-releases, candidate composes, and nightly composes. The fedfind library provides a handy interface for interacting with Fedora composes and discovering various properties of them, along with some miscellaneous helper functions. This is the Python 2 library package.
