nullmodem

A utility to loopback pseudo-terminals

  • Builds
  • Updates
  • Bugs
  • Sources

  • Koschei

nullmodem creates a virtual network of pseudo-terminals. It can be used as an adapter to connect two programs that normally need serial interface cards.
