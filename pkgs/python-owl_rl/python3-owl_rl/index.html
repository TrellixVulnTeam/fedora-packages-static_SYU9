A simple implementation of the OWL2 RL Profile, as well as a basic RDFS inference, on top of RDFLib. Based mechanical forward chaining. The distribution contains: * scripts/RDFConvertService: can be used as a CGI script to invoke the library. It may have to be adapted to the local server setup. * scripts/owlrl: script that can be run locally on to transform a file into RDF (on the standard output). Run the script with -h to get the available flags.
