nbtscan

Tool to gather NetBIOS info from Windows networks

  • Builds
  • Updates
  • Bugs
  • Sources

  • Koschei

NBTscan is a program for scanning IP networks for NetBIOS name information. It sends a NetBIOS status query to each address in supplied range and lists received information in human readable form.
