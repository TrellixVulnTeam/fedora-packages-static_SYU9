php-pclzip

Compression and extraction functions for Zip formatted archives

  • Koji
  • SCM

PclZip library offers compression and extraction functions for Zip formatted archives (WinZip, PKZIP). PclZip gives you the ability to manipulate zip formatted archives. You can create an archive, list the content and extract all its content in the file system. PclZip defines an object class which represent a Zip Archive. This class manages the archive properties and offers access method and actions on the archive.
