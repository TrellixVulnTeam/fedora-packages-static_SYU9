perl-CGI-Application-Plugin-Forward

Pass control from one run mode to another in CGI::Application

  • Builds
  • Updates
  • Bugs
  • Sources
  • Crash Reports
  • Koschei

This module gives CGI::Application a forward method that passes control to another run mode and returns its output. This is equivalent to calling $self->$other_runmode, except that CGI::Application's internal value of the current run mode is updated.
