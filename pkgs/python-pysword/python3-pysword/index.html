A native Python reader of the SWORD Project Bible Modules. Reads SWORD bible files (not commentaries etc.) Detection of locally installed Swrod bible modules. Supports all known SWORD module formats (ztext, ztext4, rawtext, rawtext4) Read from zipped modules, like those available from http://www.crosswire.org/sword/modules/ModDisp.jsp?modType=Bibles Cleans the extracted text of OSIS, GBF or ThML tags. Supports both python 2 and 3 (tested with 2.7 and 3.5)
