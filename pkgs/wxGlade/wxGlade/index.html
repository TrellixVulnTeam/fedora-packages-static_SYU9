wxGlade

A wxWidgets/wxPython/wxPerl GUI designer

wxGlade is a GUI designer written in Python with the popular GUI toolkit wxPython, that helps you create wxWidgets/wxPython user interfaces. At the moment it can generate Python, C++, Perl and XRC (wxWidgets' XML resources) code. Starting from version 0.7.0, wxGlade requires wxPython >= 2.8 and Python >= 2.4 and <= 2.7.
