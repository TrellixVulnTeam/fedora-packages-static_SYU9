gnome-chemistry-utils

A set of chemical utilities

This is a meta-package for applications in the GNOME Chemistry Utils suite: * A 3D molecular structure viewer (GChem3D). * A Chemical calculator (GChemCalc). * A 2D structure editor (GChemPaint). * A periodic table of the elements application (GChemTable). * A crystalline structure editor (GCrystal). * A spectra viewer (GSpectrum).
