This package is a set of chemical utils. Several programs are available: * A 3D molecular structure viewer (GChem3D). * A Chemical calculator (GChemCalc). * A 2D structure editor (GChemPaint). * A periodic table of the elements application (GChemTable). * A crystalline structure editor (GCrystal). * A spectra viewer (GSpectrum). This package contains a plugin adding a few chemistry-related functions to gnumeric.
