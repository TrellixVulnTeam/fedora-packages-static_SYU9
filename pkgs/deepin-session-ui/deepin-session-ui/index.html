This project include those sub-project: - dde-shutdown: User interface of shutdown. - dde-lock: User interface of lock screen. - dde-lockservice: The back-end service of locking screen. - lightdm-deepin-greeter: The user interface when you login in. - dde-switchtogreeter: The tools to switch the user to login in. - dde-lowpower: The user interface of reminding low power. - dde-osd: User interface of on-screen display. - dde-hotzone: User interface of setting hot zone.
