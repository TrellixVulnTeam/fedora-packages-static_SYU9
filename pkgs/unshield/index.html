unshield

Install InstallShield applications on a Pocket PC

This tool allows the extraction of InstallShield format cabinet files (which are different from Microsoft cabinet files). It was initially developed as a part of the SynCE project to aid with installing applications for Pocket PC devices, which were often contained in InstallShield installers, but these days that is rather less likely to be the primary use case.
