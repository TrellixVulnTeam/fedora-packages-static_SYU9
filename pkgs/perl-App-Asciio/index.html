This gtk2-perl application allows you to draw ASCII diagrams in a modern (but simple) graphical application. The ASCII graphs can be saved as ASCII or in a format that allows you to modify them later. This package contains the back-end libraries needed to implement asciio. For the actual application itself, please install the 'asciio' package.
