Alnuth is an extension for the computer algebra system GAP and forms part of a standard installation. The functionality of Alnuth lies in ALgebraic NUmber THeory. It provides an interface from GAP to certain number theoretic functions from the computer algebra system PARI/GP. Most computations with Alnuth rely on this interface.
