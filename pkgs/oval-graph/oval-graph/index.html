oval-graph

Tool for visualization of SCAP rule evaluation results

  • Koschei

Oval_graph is a tool that displays the results of evaluating SCAP rules. In the form of a tree according to the OVAL standard. Using the `arf-to-graph` command, you can simply view the result of rule. Use `arf-to-json` to generate a rule result in json. Using the `json-to-graph` command, you can view the results of rules from json file.
