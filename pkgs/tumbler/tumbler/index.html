D-Bus service for applications to request thumbnails

Tumbler is a D-Bus service for applications to request thumbnails for various URI schemes and MIME types. It is an implementation of the thumbnail management D-Bus specification described on http://live.gnome.org/ThumbnailerSpec written in an object-oriented fashion Additional thumbnailers can be found in the tumbler-extras package
