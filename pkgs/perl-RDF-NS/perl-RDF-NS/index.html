perl-RDF-NS

Popular RDF name space prefixes from prefix.cc

  • Builds
  • Updates
  • Bugs
  • Sources
  • Crash Reports
  • Koschei

Hard-coding URI name spaces and prefixes for RDF applications is neither fun nor maintainable. In the end we all use more or less the same prefix definitions, as collected at <http://prefix.cc/>. This Perl module includes all these prefixes as defined at specific snapshots in time.
