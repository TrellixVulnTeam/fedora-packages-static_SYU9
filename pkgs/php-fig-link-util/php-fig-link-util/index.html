php-fig-link-util

Common utility implementations for HTTP links

This package includes common utilities to assist with implementing PSR-13 [1]. Note that it is not intended as a complete PSR-13 implementation, only a partial implementation to make writing other implementations easier. Autoloader: /usr/share/php/Fig/Link/autoload.php [1] http://www.php-fig.org/psr/psr-13/
