gnome-shell-extension-appindicator

AppIndicator/KStatusNotifierItem support for GNOME Shell

  • Builds

This extension integrates Ubuntu AppIndicators and KStatusNotifierItems (KDE's blessed successor of the systray) into GNOME Shell. You can use gnome-tweaks (additional package) or run in terminal: $ gnome-extensions enable appindicatorsupport@rgcjonas.gmail.com
