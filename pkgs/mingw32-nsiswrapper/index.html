Helper program for making NSIS Windows installers

NSISWrapper is a helper program for making Windows installers, particularly when you are cross-compiling from Unix. NSIS (a separate package) is a program for building Windows installers. This wrapper simply makes it easier to generate the installer script that NSIS needs.
