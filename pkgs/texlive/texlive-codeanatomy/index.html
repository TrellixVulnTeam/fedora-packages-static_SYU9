The idea of this Package is to typeset illustrations of pieces of code with annotations on each single part of code (Code Anatomy). The origin of this idea are code illustrations from the book "Computer Science: An Interdisciplinary Approach" from Robert Sedgewick and Kevin Wayne. The package depends on expl3, xparse, and TikZ.
