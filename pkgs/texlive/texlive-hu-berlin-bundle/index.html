This package provides files according to the corporate design of the Humboldt-Universitat zu Berlin. This is not an official package by the university itself, and not officially approved by it. More information can be found in the Humboldt University's corporate design guideline and on the website https://www.hu-berlin.de/de/hu-intern/design. At present, the bundle contains a letter class based on scrlttr2 and a package hu-berlin-base.sty which contains all relevant code for documents and documentclasses of the bundle.
