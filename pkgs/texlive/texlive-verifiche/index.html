texlive-verifiche Subpackage of texlive

A LaTeX package to typeset (Italian) high school tests

  • Builds
  • Updates
  • Bugs
  • Sources
  • Crash Reports
  • Koschei

The purpose of this package is to manage the exercises for a test, their points, levels of difficulty, and solutions. Some typical formats of exercises are already implemented: Plain exercise "Complete the Text" "True or false" Closed questions Open questions "Find the error"
