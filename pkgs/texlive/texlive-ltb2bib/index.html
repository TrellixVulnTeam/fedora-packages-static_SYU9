texlive-ltb2bib Subpackage of texlive

converts amsrefs' .ltb bibliographical databases to BibTeX format

This package implements a LaTeX command that converts an amsrefs bibliographical database (.ltb) to a BibTeX bibliographical database (.bib). ltb2bib is the reverse of the "amsxport" option in amsrefs. Typical uses are: produce bib entries for some publishers which don't accept amsrefs (Taylor & Francis, for example); import an ltb database in a database management program, e.g. for sorting; access one's ltb database within emacs's RefTeX mode.
