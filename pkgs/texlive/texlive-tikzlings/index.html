A collection of LaTeX packages for drawing cute little animals and similar creatures using TikZ. Currently, the following TikZlings are included: anteater bear bee cat chicken coati elephant hippo koala marmot mole mouse owl panda penguin pig rhino sheep sloth snowman squirrel These little drawings can be customized in many ways.
