Qutepart is a code editor widget for PyQt. Features: - Syntax highlighting for 196 languages. - Smart indentation for many languages. - Line numbers. - Bookmarks. - Advanced edit operations. - Matching braces highlighting. - Autocompletion based on document content. - Marking too long lines with red line. - Rectangular selection and copy-paste. - Linter marks support.
