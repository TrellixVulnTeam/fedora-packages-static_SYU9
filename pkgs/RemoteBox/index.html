RemoteBox

VirtualBox is traditionally considered to be a virtualization solution aimed at the desktop as opposed to other solutions such as KVM, Xen and VMWare ESX which are considered more server oriented solutions. While it is certainly possible to install VirtualBox on a server, it offers few remote management features beyond using the vboxmanage command line. RemoteBox aims to fill this gap by providing a graphical VirtualBox client which is able to communicate with and manage a VirtualBox server installation. RemoteBox achieves this by using the vboxwebsrv feature of VirtualBox that allows its API to be accessed using a protocol called SOAP, even across a network.
