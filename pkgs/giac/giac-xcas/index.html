giac-xcas Subpackage of giac

GUI application for Giac

Xcas is the Fltk graphic user interface to the computer algebra system giac. It supports formal computations, interactive 2D geometry, 3D plotting, spreadsheets with formal calculus and a Logo mode. There is also a programming editor, and many ways to consult the html help.
