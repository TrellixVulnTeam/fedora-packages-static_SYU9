perl-HTML-BarGraph

Generate multiset bar graphs using plain HTML

HTML::BarGraph is a module that creates graphics for one or more data-sets, using plain HTML and, optionally, one-pixel images, which are stretched using the width and height attributes of the HTML img tag.
