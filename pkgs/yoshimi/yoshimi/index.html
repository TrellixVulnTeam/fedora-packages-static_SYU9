Yoshimi is a rewrite of ZynAddSubFx to improve its compatibility with the Jack Audio Connection Kit. ZynAddSubFX is an open source software synthesizer capable of making a countless number of instrument sounds. It is microtonal, and the instruments made by it sound like those from professional keyboards. The program has effects like Reverb, Echo, Chorus, Phaser...
