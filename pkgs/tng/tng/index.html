Trajectory Next Generation binary format manipulation library

TRAJNG (Trajectory next generation) is a program library for handling molecular dynamics (MD) trajectories. It can store coordinates, and optionally velocities and the H-matrix. Coordinates and velocities are stored with user-specified precision. In addition, program specific information (text strings) can optionally be stored in the beginning of each file. Atomic labels can also optionally be stored once in the beginning of the file.
