Pure Go library for generating, decoding and encrypting JSON Web Tokens. Zero dependency, relies only on standard library. Supports full suite of signing, encryption and compression algorithms defined by JSON Web Algorithms as of July 4, 2014 version. Extensively unit tested and cross tested (100+ tests) for compatibility with jose.4.j, Nimbus-JOSE-JWT, json-jwt and jose-jwt libraries.
