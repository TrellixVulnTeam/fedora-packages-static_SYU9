drupal7-footnotes

Add automatically numbered footnotes to your posts

Footnotes module can be used to easily create automatically numbered footnote references into an article or post (such as a reference to a URL). It now supports also TinyMCE and CKEditor via WYSIWYG module. This package provides the following Drupal modules: * footnotes * footnotes_field * footnotes_wysiwyg * footnotes_views
