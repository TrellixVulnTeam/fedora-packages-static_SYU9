X2Go is a server based computing environment with: - session resuming - low bandwidth support - session brokerage support - client side mass storage mounting support - audio support - authentication by smartcard and USB stick This Python module allows you to integrate X2Go client support into your Python applications by providing a Python-based X2Go client API.
