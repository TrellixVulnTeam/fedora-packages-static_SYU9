perl-MooseX-App-Cmd

Mashes up MooseX::Getopt and App::Cmd

This package marries App::Cmd with MooseX::Getopt. Use it like the App::Cmd man-page advises (especially see the App::Cmd::Tutorial man-page), swapping App::Cmd::Command for MooseX::App::Cmd::Command. Then you can write your commands as Moose classes, with the MooseX::Getopt defining the options for you instead of 'opt_spec' returning a Getopt::Long::Descriptive spec.
