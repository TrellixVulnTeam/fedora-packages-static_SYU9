The Doctrine Reflection project is a simple library used by the various Doctrine projects which adds some additional functionality on top of the reflection functionality that comes with PHP. It allows you to get the reflection information about classes, methods and properties statically. Autoloader: /usr/share/php/Doctrine/Common/Reflection/autoload.php
