nodeunit

Easy asynchronous unit testing framework for Node.js

Nodeunit provides an easy asynchronous unit testing framework for Node.js: - helps you avoid common pitfalls when testing asynchronous code - easy to add test cases with setUp and tearDown functions if you wish - flexible reporters for custom output - built-in support for HTML and jUnit XML - allows the use of mocks and stubs This package ship nodeunit-x, the maintained release for the gruntjs project, as it is more active than the original upstream project.
