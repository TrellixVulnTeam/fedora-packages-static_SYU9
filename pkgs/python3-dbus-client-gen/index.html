Library for Generating D-Bus Client Code

This library contains a few methods that consume an XML specification of a D-Bus interface and return classes or functions that may be useful in constructing a python D-Bus client. The XML specification has the format of the data returned by the Introspect() method of the Introspectable interface. Python 3 version.
