kBuild

A cross-platform build environment

  • Bodhi
  • Bugzilla
  • FAF
  • Koji

This is a GNU make fork with a set of scripts to simplify complex tasks and portable versions of various UNIX tools to ensure cross-platform portability. It is used mainly to build VirtualBox packages for RPM Fusion repository.
