perl-MooseX-Types-URI

URI related types and coercions for Moose

  • Bodhi
  • Bugzilla
  • FAF
  • Koji
  • SCM

This package provides Moose types for fun (and profit) with the URI classes. It has slightly DWIMier types than the the URI classes have due to implementation details, so the types should be more forgiving when ducktyping will work anyway (e.g. URI::WithBase does not inherit URI).
