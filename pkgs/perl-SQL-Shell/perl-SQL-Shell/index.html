SQL::Shell is a command-interpreter API for building shells and batch scripts. A command-line interface with readline support is included as part of the CPAN distribution. See SQL::Shell::Manual for a user guide. SQL::Shell offers features similar to the mysql or sql*plus client programs but is database independent. This package provides the backend SQL::Shell libraries. For the command-line interpreter (sqlsh), please also install the sqlsh package.
