lxappearance-obconf

Plugin to configure Openbox inside LXAppearance

  • Builds
  • Updates
  • Bugs
  • Sources

This plugin adds an additional tab called "Window Border" to LXAppearance. It is only visible when the plugin is installed and Openbox is in use.
