perl-Directory-Queue

Object oriented interface to a directory based queue

  • Builds
  • Updates
  • Bugs
  • Sources
  • Crash Reports
  • Koschei

The goal of this module is to offer a simple queue system using the underlying file system for storage, security and to prevent race conditions via atomic operations. It focuses on simplicity, robustness and scalability.
