badwolf

Web Browser which aims at security and privacy over usability

BadWolf is a minimalist and privacy-oriented WebKitGTK+ browser. - Privacy-oriented: No browser-level tracking, multiple ephemeral isolated sessions per new unrelated tabs, JavaScript off by default. - Minimalist: Small codebase (~1 500 LoC), reuses existing components when available or makes it available. - Customizable: WebKitGTK native extensions, Interface customizable through CSS. - Powerful & Usable: Stable User-Interface; The common shortcuts are available (and documented), no vi-modal edition or single-key shortcuts are used. - No annoyances: Dialogs are only used when required (save file, print, â€¦), javascript popups open in a background tab.
