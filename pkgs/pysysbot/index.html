pysysbot

A simple python jabber bot for getting system information

  • Bodhi
  • Bugzilla
  • FAF
  • Koji
  • SCM

This python jabber (XMPP) bot is based on the jabberbot framework. The bot is capable to display details about the system it is running on. If you don't want or can stay connected through SSH all the time this is an easy way to get information about the remote system.
