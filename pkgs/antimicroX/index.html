antimicroX is a graphical program used to map keyboard keys and mouse controls to a gamepad. This program is useful for playing PC games using a gamepad that do not have any form of built-in gamepad support. antimicroX is a fork of AntiMicro which was inspired by QJoyPad but has additional features.
