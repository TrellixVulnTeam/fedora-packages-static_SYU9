This is the bindings module of the GNAT Components Collection. It provides bindings to GMP, Iconv, Readline and Syslog. The Python binding is excluded until it works with Python 3. The gnatcoll-bindings-devel package contains source code and linking information for developing applications that use the GNAT Components Collection bindings.
