JSqueeze shrinks / compresses / minifies / mangles Javascript code. It's a single PHP class that is developed, maintained and thoroughly tested since 2003 on major JavaScript frameworks (e.g. jQuery). JSqueeze operates on any parse error free JavaScript code, even when semi-colons are missing. In term of compression ratio, it compares to YUI Compressor and UglifyJS. Autoloader: /usr/share/php/Patchwork/autoload-jsqueeze.php
