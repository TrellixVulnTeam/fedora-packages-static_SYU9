gammaray-qt5 Subpackage of gammaray

Qt 5 probe for GammaRay

  • Bodhi

Provides a Qt 5 probe for GammaRay that allows introspecting Qt 5 applications. This probe is installed by default. It is possible to install probes for different architectures as well, GammaRay will then be able to inspect those applications too.
