Displays key presses on screen on supported Wayland compositors (requires wlr_layer_shell_v1 support eg sway). Usage wshowkeys [-b|-f|-s #RRGGBB[AA]] [-F font] [-t timeout] [-a top|left|right|bottom] [-m margin] [-o output] -b #RRGGBB[AA]: set background color -f #RRGGBB[AA]: set foreground color -s #RRGGBB[AA]: set color for special keys -F font: set font (Pango format, e.g. 'monospace 24') -t timeout: set timeout before clearing old keystrokes -a top|left|right|bottom: anchor the keystrokes to an edge. May be specified twice. -m margin: set a margin (in pixels) from the nearest edge -o output: request wshowkeys is shown on the specified output (unimplemented)
