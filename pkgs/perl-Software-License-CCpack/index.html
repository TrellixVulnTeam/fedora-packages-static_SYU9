perl-Software-License-CCpack

Software::License pack for Creative Commons' licenses

  • Koji
  • SCM

This "license pack" contains all of the licenses from Creative Commons, except for CC0, which is already included in Software::License. Note that I don't recommend using these licenses for your own CPAN modules (most of the licenses aren't even compatible with CPAN). However, S:L modules are useful for more than mere CPAN::Meta::license declaration, so these modules exist for those other purposes.
