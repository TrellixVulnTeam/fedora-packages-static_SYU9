Procyon is a suite of Java metaprogramming tools focused on code generation and analysis. It includes the following libraries: Core Framework, Reflection Framework, Expressions Framework, Compiler Toolset (Experimental), Java Decompiler. The Procyon libraries are available from Maven Central under group ID org.bitbucket.mstrobel.
