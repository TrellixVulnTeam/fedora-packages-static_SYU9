lua-socket

Network support for the Lua language

  • SCM

LuaSocket is a Lua extension library that is composed by two parts: The C core that provides support for the TCP and UDP transport layers, and the set of Lua modules that add support for functionality commonly needed by applications that deal with the Internet. Among the support modules, the most commonly used implement the SMTP, HTTP and FTP. In addition there are modules for MIME, URL handling and LTN12.
