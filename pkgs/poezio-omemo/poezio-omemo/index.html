poezio-omemo

OMEMO plugin for the Poezio XMPP client

  • Builds
  • Updates

  • Koschei

This plugin provides OMEMO support for Poezio client. OMEMO is an extension of the XMPP protocol defined as XEP-0384. It provides multi-end to multi-end encryption, allowing messages to be synchronized securely across multiple clients, even if some of them are offline.
