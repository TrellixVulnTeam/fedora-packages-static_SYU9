ncclient is a Python library that facilitates client-side scripting and application development around the NETCONF protocol. ncclient was developed by Shikar Bhushan.net). It is now maintained by Leonidas Poulopoulos (@leopoul) and Einar Nilsen-Nygaard (@einarnn). Docs: http://ncclient.readthedocs.org PyPI: https://pypi.python.org/pypi/ncclient
