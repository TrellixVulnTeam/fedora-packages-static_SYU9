GitLab support for vim-fugitive plugin. Enables :Gbrowse from fugitive.vim to open GitLab URLs. Sets up :Git to use hub if installed rather than git (when available). In commit messages, GitLab issues, issue URLs, and collaborators can be omni-completed (<C-X><C-O>, see :help compl-omni).
