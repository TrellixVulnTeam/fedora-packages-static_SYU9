Mailwatch is a plugin for the Xfce 4 panel. It is intended to replace the current (4.0, 4.2) mail checker plugin in Xfce 4.4. It supports IMAP and POP, local mailboxes in Mbox, Maildir and MH-Maildir format as well as Gmail.
