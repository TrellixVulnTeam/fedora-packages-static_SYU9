Emacs mode for editing OCaml source code

This package provides a caml-mode for Emacs, for editing OCaml programs, as well as an inferior-caml-mode, to run a toplevel. Caml-mode supports indentation, compilation and error retrieving, and sending phrases to the toplevel. There is also support for hilit, font-lock and imenu.
