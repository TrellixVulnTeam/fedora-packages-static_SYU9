libarcus-lulzbot

Arcus library contains C++ code and Python 3 bindings for creating a socket in a thread and using this socket to send and receive messages based on the Protocol Buffers library. It is designed to facilitate the communication between Cura and its backend and similar code. This is the Lulzbot fork.
