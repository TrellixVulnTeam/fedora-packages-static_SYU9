xrdhttpvoms

VOMS security extractor plugin for XrdHTTP

  • Builds

This package provides the VOMS security extractor plugin for XrdHTTP. Given a working setup of XrdHTTP, this library will be able to extract the VOMS information from the certificate presented by the client. The extracted information will be available for other plugins to apply authorization rules.
