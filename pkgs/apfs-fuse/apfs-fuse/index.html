apfs-fuse

A read-only FUSE driver for Apple's APFS

  • Builds

apfs-fuse is a read-only driver for the new Apple File System, APFS. Since Apple didn't document the disk format, this driver should be considered experimental. Not all compression methods are supported yet, thus the driver may return compressed files instead of uncompressed ones.
