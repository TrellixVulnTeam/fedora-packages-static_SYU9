tomcat-jsvc Subpackage of tomcat

Apache jsvc wrapper for Apache Tomcat as separate service

  • Builds
  • Updates

Systemd service to start tomcat with jsvc, which allows tomcat to perform some privileged operations (e.g. bind to a port < 1024) and then switch identity to a non-privileged user.
