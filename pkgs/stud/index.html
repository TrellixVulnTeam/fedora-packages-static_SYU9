stud

The Scalable TLS Unwrapping Daemon

stud is a network proxy that terminates TLS/SSL connections and forwards the unencrypted traffic to some backend. It is designed to handle tens of thousands of connections efficiently on multicore machines. stud has very few features -- it is designed to be paired with an intelligent backend like haproxy or nginx.
