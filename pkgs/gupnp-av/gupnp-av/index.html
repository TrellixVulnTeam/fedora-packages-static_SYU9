gupnp-av

A collection of helpers for building UPnP AV applications

GUPnP is an object-oriented open source framework for creating UPnP devices and control points, written in C using GObject and libsoup. The GUPnP API is intended to be easy to use, efficient and flexible. GUPnP-AV is a collection of helpers for building AV (audio/video) applications using GUPnP.
