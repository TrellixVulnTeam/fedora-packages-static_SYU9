perl-Ace

Perl module for interfacing with ACE bioinformatics databases

AcePerl is a Perl interface for the ACEDB object-oriented database. Designed specifically for use in genome sequencing projects, ACEDB provides powerful modeling and management services for biological and laboratory data.
