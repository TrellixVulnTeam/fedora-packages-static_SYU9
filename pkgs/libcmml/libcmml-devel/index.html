Libcmml is a library which enables the handling of documents written in CMML (Continuous Media Markup Language) for the Continuous Media Web (CMWeb). It provides a very simple API for reading files marked up with the Continuous Media Markup Language (CMML), and returns C structures containing this information in a format which can be used by an Annodexer for creating ANNODEX(tm) format documents (ANX). This package contains the header files and documentation needed for development using libcmml.
