perl-Web-Paste-Simple

Simple PSGI-based pastebin-like web site

Web::Paste::Simple is a lightweight PSGI application for operating a pastebin-like web site. It provides syntax highlighting via the CodeMirror JavaScript library. It should be fast enough for deployment via CGI.
