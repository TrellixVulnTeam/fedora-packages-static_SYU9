This package provides a library that implements run-time support for encoding (marshaling), decoding (unmarshaling), and accessing protocol buffers in the Go language. Install golang-googlecode-goprotobuf for the related protocol compiler plugin. This package contains library source intended for building other packages which use import path with github.com/golang/protobuf prefix.
