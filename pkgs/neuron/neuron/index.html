A flexible and powerful simulator of neurons and networks

NEURON is a simulation environment for modeling individual neurons and networks of neurons. It provides tools for conveniently building, managing, and using models in a way that is numerically sound and computationally efficient. It is particularly well-suited to problems that are closely linked to experimental data, especially those that involve cells with complex anatomical and biophysical properties. This package does not include MPI support. Please install the neuron-devel package to compile nmodl files and so on.
