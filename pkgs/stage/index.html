stage

A 2.5D multi-robot simulator

Stage is a fast and scalable 2.5D multiple robot simulator from the Player project. Stage can be used to simulate sensors and actuators in a low-fidelity bit-mapped environment. Stage models can be controlled with the Stage C API, or through the Player server via a Player plug-in library.
