Tkabber is a Free and Open Source client for the Jabber instant messaging system. It's written in Tcl/Tk, and works on many platforms. The choice of Tcl/Tk for a Jabber client is three-fold: * it is portable: once you install a Tcl/Tk interpreter on your system, the Tkabber script "just runs" â€” without having to compile anything; * it is customizable: Tkabber reads a configuration file when it starts that tells it the settings of various parameters; and, * it is extensible: the configuration file is actually a Tcl script, so you can replace or augment entire portions of Tkabber (if you're so inclined).
