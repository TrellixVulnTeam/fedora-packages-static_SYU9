perl-HTML-Restrict

Perl module to strip unwanted HTML tags and attributes

  • Builds
  • Updates
  • Bugs
  • Sources
  • Crash Reports
  • Koschei

A perl module that uses HTML::Parser to strip HTML from text in a restrictive manner. By default all HTML is restricted, but default behaviour may be altered by supplying own tag rules.
