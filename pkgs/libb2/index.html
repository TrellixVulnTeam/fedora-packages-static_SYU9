C library providing BLAKE2b, BLAKE2s, BLAKE2bp, BLAKE2sp. BLAKE2 is a cryptographic hash function faster than MD5, SHA-1, SHA-2, and SHA-3, yet is at least as secure as the latest standard SHA-3.
