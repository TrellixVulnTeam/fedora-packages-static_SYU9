php-seld-cli-prompt

Allows you to prompt for user input on the command line

While prompting for user input using fgets() is quite easy, sometimes you need to prompt for sensitive information. In these cases, the characters typed in by the user should not be directly visible, and this is quite a pain to do in a cross-platform way. Autoloader: /usr/share/php/Seld/CliPrompt/autoload.php
