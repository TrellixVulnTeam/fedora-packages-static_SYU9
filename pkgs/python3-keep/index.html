Personal shell command keeper and snippets manager - Save a new command with a brief description - Search the saved commands using powerful patterns - Save the commands as a secret GitHub gist - Use `keep push` and `keep pull` to sync the commands between GitHub gist and other computers. **ProTip : Save the commands you usually forget in ssh sessions and sync it with your local machine.**
