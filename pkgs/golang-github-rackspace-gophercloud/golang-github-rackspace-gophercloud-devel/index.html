Gophercloud is a flexible SDK that allows you to consume and work with OpenStack clouds in a simple and idiomatic way using Golang. Many services are supported, including Compute, Block Storage, Object Storage, Networking, and Identity. Each service API is backed with getting started guides, code samples, reference documentation, unit tests and acceptance tests.
