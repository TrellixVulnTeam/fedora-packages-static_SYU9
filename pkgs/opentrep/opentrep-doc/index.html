opentrep-doc Subpackage of opentrep

This package contains HTML pages for opentrep. All that documentation is generated thanks to Doxygen (https://doxygen.org). The content is the same as what can be browsed online (https://opentrep.sourceforge.net). Note that the PDF form of the reference manual is mainly available online (https://opentrep.sourceforge.net/refman.pdf), as the one present in that package is usually corrupted: it depends on the building conditions, and it is therefore not reliable.
