writerperfect is a collection of command-line tools to transform various document formats into EPUB. Among supported input formats are: * AbiWord * Adobe PageMaker * Apple Keynote * Corel WordPerfect * CorelDRAW * Microsoft Publisher * Microsoft Visio * Microsoft Works
