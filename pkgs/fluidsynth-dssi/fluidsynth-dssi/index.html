This is an implementation of the FluidSynth soundfont-playing software synthesizer as a DSSI plugin. It makes use of DSSI's run_multiple_synths() interface to allow sharing of resources between multiple plugin instances -- soundfont data is shared between instances, and FluidSynth's usual voice allocation methods are applied across multiple instances as if each were a FluidSynth channel.
