lxsession

Lightweight X11 session manager

LXSession is a standard-compliant X11 session manager with shutdown/ reboot/suspend support via systemd. In connection with gdm it also supports user switching. LXSession is derived from XSM and is developed as default X11 session manager of LXDE, the Lightweight X11 Desktop Environment. Though being part of LXDE, it's totally desktop-independent and only has few dependencies.
